# <a name="delete-a-sharepoint-team-site"></a>SharePoint 팀 사이트를 삭제 합니다.

1. 사이트로 이동 합니다.
    
2. 오른쪽 위에 있는 **설정** 아이콘을 클릭 합니다. 
    
3. **사이트** 설정을 클릭 합니다. **사이트 정보** 또는 **사이트 작업**아래에서 **이 사이트 삭제**를 클릭 한 다음 **삭제**를 클릭 합니다.
