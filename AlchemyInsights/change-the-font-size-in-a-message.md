# <a name="change-the-font-size-in-a-message"></a>메시지의 글꼴 크기 변경

1. 이미 메시지를 입력 한 경우 텍스트를 선택 합니다.
2. 메시지 아래에 있는  **형식** 아이콘 (첫 번째 아이콘)을 선택 합니다.
3. 메시지 위의  **글꼴 크기**  아이콘 (일곱 번째 아이콘)을 선택 합니다.
4. 크기를 선택 합니다.
