# <a name="move-files-in-sharepoint-and-onedrive"></a>SharePoint 및 OneDrive에서 파일 이동

웹 브라우저 자체에서 SharePoint 및 OneDrive 내의 다른 위치로 파일을 이동할 수 있습니다. [파일 이동 또는 복사](https://support.microsoft.com/office/move-or-copy-files-in-sharepoint-00e2f483-4df3-46be-a861-1f5f0c1a87bc?ui=en-US&rs=en-US&ad=US)를 참조 하세요.


> [!NOTE]
> SharePoint Online의 클래식 환경을 사용 하는 경우 **에는 이동을** 사용할 수 없습니다.
