# <a name="use-your-browser-to-check-spelling"></a>브라우저를 사용 하 여 맞춤법 검사

Outlook.com에는 맞춤법 검사 옵션이 없습니다. 맞춤법을 검사 하려면 웹 브라우저를 사용 해야 합니다. Microsoft Edge, Internet Explorer 10 이상 버전 및 현재 버전의 Firefox, Chrome 및 Safari에서 맞춤법 검사를 사용할 수 있습니다. 맞춤법 검사를 확인 하는 방법에 대 한 자세한 내용은 웹 브라우저의 옵션을 확인 하세요.
