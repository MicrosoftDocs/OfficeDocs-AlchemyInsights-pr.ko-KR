# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>흐름의 조건이 나 고급 수식을 사용 하 여

조건이 true 인 경우에 작업을 수행 하는 흐름을 사용자 지정할 수 있습니다. 정보, [흐름에 조건 추가](https://go.microsoft.com/fwlink/?linkid=872112)참조 하십시오.
  
고급 수식을 만들려면 [Azure 논리 앱에 대 한 워크플로 정의 언어 스키마](https://aka.ms/logicexpressions)를 참조 하십시오.
