# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>흐름에 조건 또는 고급 수식 사용

조건이 true 인 경우에만 작업을 수행 하도록 흐름을 사용자 지정할 수 있습니다. 자세한 내용은 [흐름에 조건 추가](https://go.microsoft.com/fwlink/?linkid=872112)를 참조 하십시오.
  
고급 수식을 만들려면 [Azure 로직 앱 용 워크플로 정의 언어 스키마](https://aka.ms/logicexpressions)를 참조 하세요.
