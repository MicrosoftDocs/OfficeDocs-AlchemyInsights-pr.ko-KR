# <a name="share-a-file-or-folder-in-onedrive-or-sharepoint"></a>OneDrive 또는 SharePoint에서 파일 또는 폴더 공유

1. 공유할 파일 또는 폴더를 찾아서 선택한 다음 **공유**를 클릭 합니다.
    
2. 반드시 연결 보내기 대화 상자에서 아래쪽 화살표를 클릭 하 여 연결 형식을 변경 합니다. **이 링크를 사용**하는 사용자는 누구 입니까?에서 링크에 액세스할 수 있는 사람을 변경 합니다. 사용자가 파일을 볼 수는 있지만 편집 하지 못하게 하려면 **편집 허용** 상자를 선택 취소 합니다. 
    
    **적용**을 클릭합니다.
    
    > [!NOTE]
    > 원하는 공유 옵션을 사용할 수 없는 경우 조직 및 사이트 모음에 대해 허용 되는지 확인 해야 합니다. 이렇게 하려면 [SharePoint Online에 대 한 외부 공유 설정 또는 해제](https://go.microsoft.com/fwlink/?linkid=866426)를 참조 하세요. 
  
3. 공유할 사용자의 이름을 입력 하 고 **보내기를**클릭 합니다.
