# <a name="unable-to-delete-photos-from-files"></a>파일에서 사진을 삭제할 수 없습니다.

파일 또는 사진을 삭제:
  
1. 사용자의 **받은 편지함** 또는 **보낸 편지함** 폴더로 이동 하 고에 연결 된 전자 메일 메시지를 삭제 합니다. 
    
2. **삭제 된 항목**에서 항목을 제거 합니다. 
    
자세한 내용은 [파일 또는 Outlook.com에서 사진을 삭제 하는 방법을](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx)참조 하세요.
