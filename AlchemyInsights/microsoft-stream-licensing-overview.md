# <a name="microsoft-stream-licensing-overview"></a>Microsoft Stream License 개요

Microsoft Stream은 사용자 당 구독을 기준으로 사용이 허가 된 클라우드 서비스입니다. 이 기능은 적합 한 Microsoft 365 구독의 일부로 사용할 수 있습니다. 각 구독 유형에 대 한 자세한 내용은 Microsoft 365 가격 책정를 참조 하세요.

- [Microsoft Stream License 개요](https://docs.microsoft.com/stream/license-overview)
