# <a name="add-phone-number"></a>전화번호 추가

전화를 걸고 받을 수 있도록 조직에서 사용자를 설정하려면 먼저 사용자의 [전화번호를 받아야](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview) 합니다. 전화 시스템 설정을 완료하는 단계별 프로세스를 제공했습니다. [조직에서 전화 시스템 설정](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview)을 확인하세요.
