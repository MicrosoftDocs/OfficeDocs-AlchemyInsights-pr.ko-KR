# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Outlook에서 웹에 공유 사서함을 열 때 사용 권한 오류가 시작?

**오류가 발생 하는 경우 오류가 발생 합니다. 이 사서함을 열 수 있는 권한이.** Outlook에서 웹에 공유 사서함을 열려고 계정을 공유 사서함에 액세스할 수 있는 권한을 권한이 없는 합니다. 조직 관리자 [공유 사서함 만들기](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9)항목의에서 지침을 수행 하 여 공유 사서함에 계정 사용 권한을 할당할 수 있습니다.
