# <a name="microsoft-viva-insights-availability"></a>Microsoft Viva Insights 가용성

Microsoft Viva Insights를 미리 보기에서 사용할 수 있습니다.

[다운로드](https://aka.ms/InsightsDocumentation) Microsoft Teams의 Viva Insights

[Viva](https://aka.ms/VivaInsights) Insights를 통해 사용자와 비즈니스가 번성할 수 있는 문화를 만드는 방법에 대해 자세히 알아보아야 합니다.

자세한 내용은 Microsoft [Viva Insights를](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/microsoft-viva-insights-helps-people-nurture-wellbeing-and-be/ba-p/2107010)통해 사람들이 웰링을 잘하고 최고의 웰링을 하게 하는 방법을 참조하세요.
