# <a name="links-in-outlookcom-dont-work-in-firefox"></a>Outlook.com에 링크가 Firefox에서 작동 하지 않습니다.

Outlook.com 전자 메일 메시지에 대 한 링크는 설치 된 Firefox 브라우저에서 본 하는 경우 작동 하지 않습니다.
  
Mozilla는 Firefox, 하지만 그때까지 [수정 방법 (영문)](https://go.microsoft.com/fwlink/p/?linkid=2001502&amp;clcid=0x409) 입니다. 
  
- [Microsoft에 지](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) 또는 다른 웹 브라우저를 사용 하십시오. 
    
- Outlook.com 베타를 사용 합니다. [Outlook.com 베타 버전에 알아보십시오.](https://go.microsoft.com/fwlink/p/?linkid=874356&amp;clcid=0x409)
    
- 읽기 창에서 전자 메일을 확인 합니다. 자세한 내용은 읽기 [Outlook.com 사서함의 모양을 변경](https://go.microsoft.com/fwlink/p/?linkid=2001401&amp;clcid=0x409)하는 방법입니다.
