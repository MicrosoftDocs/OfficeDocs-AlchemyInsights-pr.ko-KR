# <a name="solutions-for-error-30179-25"></a>오류 30179-25 해결 방법

하드 드라이브에 공간이 부족 한 경우 "Office를 스트리밍할 수 없습니다." (오류 코드 [30179-25](https://support.office.com/article/e40d3c7d-98f6-4284-94a0-882beaa44593?wt.mc_id=Alchemy_ClientDIA)) 메시지가 표시 될 수 있습니다. 
  
[하드 드라이브에 공간을 확보](https://support.microsoft.com/help/12425/windows-10-free-up-drive-space) 한 다음 [Office를 다시 설치](https://portal.office.com/OLS/MySoftware.aspx)합니다.
