# <a name="troubleshoot-issues-with-apple-mdm-push-certificate-in-microsoft-intune"></a>Microsoft Intune에서 Apple MDM 푸시 인증서 문제 해결

아래에 나열 된 리소스를 검토 하 여 지금 문제를 해결 하세요.
  
1. [이 문서](https://docs.microsoft.com/intune/apple-mdm-push-certificate-get) 를 검토 하 여 Apple MDM 푸시 인증서를 구성 하거나 갱신 합니다.

2. [Intune 문제 해결 포털](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) 을 사용 하 여 일반적인 등록 오류를 진단 하 고 해결 합니다. 자세한 내용을 보려면 [이 문서](https://docs.microsoft.com/intune/help-desk-operators) 를 검토 하세요.

3. 다음 문서를 검토 하 여 각 [문제 해결 가이드](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) 및 [문서 문제 해결](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)을 방지 하는 일반적인 오류 목록을 확인 합니다.
