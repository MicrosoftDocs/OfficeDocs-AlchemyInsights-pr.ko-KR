# <a name="change-my-language"></a>내 언어 변경

Outlook 앱은 자동으로 장치가 설정 된 언어로 번역 됩니다. 

( [Outlook에서 지원 되는 언어 목록을](https://acompli.helpshift.com/a/outlook/?s=general-questions&f=in-which-languages-is-your-app-translated)참조 하세요.) 

**장치에서 언어를 변경 하려면**: 

**Android**: 

- 장치 설정 > 언어 및 입력 

**IOS**: 

- 장치 설정 > 일반 > 언어 & 지역
