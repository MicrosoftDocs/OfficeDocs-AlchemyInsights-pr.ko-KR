# <a name="distribution-group-not-showing-in-groups-list"></a>그룹 목록에 표시 되지 메일 그룹

새 메일 그룹은 Exchange 관리 센터 센터에 표시 되기 전에 60 분까지 걸릴 수 있습니다.
  
구성원을 추가 하거나 다른 편집을 지금 바로 확인 하려는 경우 [Exchange 관리 센터](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx)의 그룹 기능을 사용 하십시오.
