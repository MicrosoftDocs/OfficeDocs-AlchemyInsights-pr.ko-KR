# <a name="sharepoint-publishing-infrastructure"></a>SharePoint 게시 인프라


<p>SharePoint에는 인트라넷의 서식 있는 웹 페이지를 작성 및 게시할 수 있는 전통적인 기능 집합이 포함 되어 있습니다. 사이트 모음 수준에서 SharePoint 게시 인프라를 활성화 하 여 설정 합니다.</p>  <p>자세한 내용을 보려면 아래 링크를 방문 하세요.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">게시 기능 사용</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">게시 기능</a></li>  </ul>  <p>참고:</p>  <ul>  <li>클래식 게시 사이트 서식 파일에는 기본적으로 게시 기능이 포함 되어 있습니다.</li>  <li>최신 사이트에서는 클래식 게시 기능을 사용할 수 없습니다. 최신 사이트는 기본적으로 새로운 대안을 제공 합니다.</li>  </ul>
