# <a name="does-visio-work-on-a-mac"></a>Visio가 Mac에서 작동 합니까?

Mac용 Visio 데스크톱 응용 프로그램은 없습니다. 그러나 앱이 필요한 사용자에 [게 Visio online 라이선스가 할당](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799)되어 있으면 웹 브라우저에서 [visio online의 다이어그램을 보고, 만들고, 편집할](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799) 수 있습니다.
