# <a name="resolving-excel-clipboard-error"></a>Excel 클립보드 오류 해결

Excel에서 "클립보드에 문제가 있지만 이 통합 문서 내에 콘텐츠를 붙여넣을 수 있습니다."라는 오류가 표시될 경우 다음 단계에 따라 Live Preview 기능을 사용 안 하게 합니다.

1. Excel에서 파일 **옵션 을**  >  **클릭합니다.**
3. Excel 옵션 **대화** 상자에서 일반 을 **선택합니다.**
4. 사용자 **인터페이스 옵션 아래에서** 실시간 **미리 보기 사용의 선택을 취소합니다.**
5. **확인** 을 클릭합니다.
6. Excel을 다시 시작합니다.
