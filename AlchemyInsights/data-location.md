# <a name="data-location"></a>데이터 위치

중복성을 위해 데이터 센터에 데이터의 여러 복사본을 유지 합니다. [데이터의 위치를 참조 하십시오](https://office.com/datamaps). 새 국가 또는 지역으로 데이터의 위치를 확장 하는 경우 한 달 사전 통지를 제공 해 드리겠습니다.
  
Office 365 보안, 프라이버시 및 규정 준수 하는 방법에 대 한 자세한 정보를 [Office 365 보안 센터](https://products.office.com/business/office-365-trust-center-welcome)를 참고 하십시오.
