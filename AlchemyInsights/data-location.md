# <a name="data-location"></a>데이터 위치

관리 센터에서 [ **조직 프로필** 페이지로](https://admin.microsoft.com/AdminPortal/Home#/Settings/OrganizationProfile) 이동 하 여 **데이터 위치**를 선택 하 여 테 넌 트의 위치를 볼 수 있습니다.

구독 하 고 있지 않은 응용 프로그램의 경우 [Microsoft 365 고객 데이터가 저장 되는 위치](https://docs.microsoft.com/office365/enterprise/o365-data-locations)를 확인 합니다.
