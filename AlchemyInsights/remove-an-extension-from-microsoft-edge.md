# <a name="remove-an-extension-from-microsoft-edge"></a>Microsoft Edge에서 확장 제거

Microsoft Edge에서 확장을 제거하려면

1. Microsoft Edge를 엽니다.
2. Select **... > Extensions**.
3. 설치된 **확장 페이지에서** 제거할 확장을 찾은 다음 옆에 있는 **제거를** 선택합니다.
4. 확인 대화 상자에서 제거를 **선택합니다.**
