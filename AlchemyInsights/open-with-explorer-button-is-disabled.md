# <a name="the-open-with-explorer-button-is-disabled"></a>탐색기 단추 열기를 사용 하지 않도록 설정

- Internet Explorer 10 또는 Internet Explorer 11에서 SharePoint 또는 OneDrive 위치로 이동 합니다. 단추는 다른 웹 브라우저와 호환 되지 않습니다.
    
> [!NOTE]
> 동기화 하는 것 중 권장 라이브러리 파일 탐색기에서 열기를 복사 하거나 후, 하지만 라이브러리에서 정기적으로 작동 하려면 여러 파일 및 폴더를 이동 해야 하는 경우에 유용 합니다. 자세한 도움말을 참고 하 고 파일 탐색기에서 열기 다른 문제를 해결 하려면 [탐색기에서 열기](https://go.microsoft.com/fwlink/?linkid=871665)를 참조 합니다. 동기화를 설정 하는 방법에 대 한 정보를 [새 OneDrive 동기화 클라이언트와 함께 동기화 SharePoint 파일](https://go.microsoft.com/fwlink/?linkid=871666)을 참조 하십시오.
