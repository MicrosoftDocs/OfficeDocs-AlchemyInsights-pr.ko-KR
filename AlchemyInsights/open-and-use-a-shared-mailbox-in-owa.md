# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>웹용 Outlook에서 공유 사서함 열기 및 사용

액세스 권한이 있는 공유 사서함은 웹의 Outlook(이전의 Outlook Web App)에 자동으로 표시되지 않습니다.

공유 사서함과 기본 사서함을 함께 표시하거나 별도의 브라우저 창에서 공유 사서함을 열 수 있습니다. 공유 사서함에 대한 새 메시지 알림을 받으려면 별도의 브라우저 창에서 공유 사서함을 열어야 합니다. 더 자세한 내용은 [Outlook Web App에서 공유 사서함 열기 및 사용](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)를 참조하십시오.
