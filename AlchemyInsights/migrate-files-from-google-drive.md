# <a name="migrate-files-from-google-drive"></a>Google Drive에서 파일 마이그레이션

Google Drive에서 SharePoint Online으로 마이그레이션하는 경우 **Mover** 를 사용해 보세요.

자세한 내용은 [G Suite Connector 허용](https://docs.microsoft.com/sharepointmigration/mover-gsuite)을 참조하세요.
