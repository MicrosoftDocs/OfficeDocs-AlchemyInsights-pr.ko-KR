# <a name="changing-public-folder-permissions"></a>공용 폴더 사용 권한 변경

사용자와 함께 Outlook에서 공용 폴더 사용 권한을 변경할 수 있습니다. 참조 [설정 공용 폴더에 대 한 사용 권한을 변경 또는](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005)합니다. 관리자가 Exchange 관리 센터에서 사용 권한을 변경할 수도 있습니다.
  
- Office 365 관리 센터에서 **관리 센터** 로 이동 \> **Exchange**합니다.
    
- **공용 폴더**를 선택 합니다.
    
여기에서 사용 권한 관리로 보안 그룹의 사용자를 할당 하 여 개별 공용 폴더에 대 한 사용 권한을 변경할 수 있습니다. 공용 폴더에 대 한 사용 권한을 변경 하려면 것의 소유자가 되도록 해야 합니다.
