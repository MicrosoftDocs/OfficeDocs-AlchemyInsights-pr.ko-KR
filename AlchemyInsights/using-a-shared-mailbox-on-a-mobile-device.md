# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>모바일 장치에서 공유 사서함 사용

모바일 장치에 설치 된 Outlook 앱에서 공유 사서함에 액세스할 수 없습니다. 이 기능을 원하시나요? Outlook 엔지니어링 팀에 게 알려 줍니다. [여기에서 투표](https://go.microsoft.com/fwlink/?linked=862116)하세요.
  
브라우저를 열고 Office 365에 로그인 한 다음 웹용 Outlook으로 이동 하 여 모바일 장치에서 공유 사서함만 사용할 수 있습니다. 자세한 내용은 [웹용 Outlook에서 공유 사서함 추가](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)를 참조 하세요.
  
모바일 장치에서 Outlook 앱을 사용 하 여 공유 사서함에 액세스 해야 하는 경우에는 공유 사서함을 사용 하는 대신 Office 365 그룹을 만드는 것이 좋습니다. 자세한 내용은 [그룹 비교](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx)를 참조 하세요.
  
자세한 내용은 [iOS 및 Android 용 Outlook에 대 한 앱 내 도움말 가져오기](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243) 를 참조 하세요.
