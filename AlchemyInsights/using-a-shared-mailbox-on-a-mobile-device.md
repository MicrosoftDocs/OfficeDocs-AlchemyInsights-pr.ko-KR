# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>모바일 장치에서 공유 사서함 사용

모바일 장치에서 공유 사서함을 사용 하려면 [outlook mobile을 다운로드](https://products.office.com/outlook-mobile-for-android-and-ios) 한 다음 [outlook mobile에 공유 사서함을 추가](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f)해야 합니다.
  
도움말을 보려면 [iOS 및 Android 용 Outlook에 대 한 앱 내 도움말 가져오기](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
