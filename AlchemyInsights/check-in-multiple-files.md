# <a name="check-in-several-files-at-once"></a>여러 파일을 한 번에 체크 인

<p>파일을 체크 인하고 체크 아웃 하는 것은 콘텐츠를 업데이트 하는 방법을 추적 하 고 제어 하는 좋은 방법입니다. 사이트에서 파일을 변경 하려는 경우 다른 사용자가 편집할 수 없도록 하려면 파일을 체크 아웃 합니다. 파일을 체크 아웃 한 경우 온라인 또는 오프 라인으로 편집 하 고 필요한 경우&mdash;여러 번 저장할 수 있습니다.</p> <p>다음은이 기능과 관련 된 몇 가지 유용한 링크입니다.</p> <ul> <li><a href="https://support.office.com/en-us/article/check-out-check-in-or-discard-changes-to-files-in-a-library-7e2c12a9-a874-4393-9511-1378a700f6de">파일 변경 내용을 체크 아웃, 체크 인 또는 삭제 하는 방법</a></li> <li><a href="https://support.office.com/en-us/article/Top-questions-about-check-out-check-in-and-versions-7E941339-E972-4C7A-A79A-80A1FCF84076">체크 아웃, 체크 인, 버전에 대해 가장 많이 묻는 질문</a></li> <li><a href="https://support.office.com/en-us/article/check-out-check-in-or-discard-changes-to-files-in-a-library-7e2c12a9-a874-4393-9511-1378a700f6de">한 번에 여러 파일 체크 인</a></li> <li><a href="https://support.office.com/en-us/article/enable-and-configure-versioning-for-a-list-or-library-1555d642-23ee-446a-990a-bcab618c7a37?ocmsassetID=HA102772148&amp;CTT=3&amp;CorrelationId=52441bb1-a619-4375-89d5-19d28769890f&amp;ui=en-US&amp;rs=en-US&amp;ad=US">라이브러리에 대해 버전 관리/체크 아웃 사용 또는 사용 안 함</a></li> </ul> <p>&nbsp;</p>
