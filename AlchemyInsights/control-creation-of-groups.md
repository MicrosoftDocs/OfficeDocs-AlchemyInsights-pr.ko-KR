# <a name="control-creation-of-microsoft-365-groups"></a>Microsoft 365 그룹의 컨트롤 만들기

기본적으로 회사 내의 모든 사용자는 그룹을 만들 수 있습니다. 필요한 경우이를 변경 하 고 회사 내 사용자 집합에 그룹을 만들 수 있는 권한을 할당할 수 있습니다. 이렇게 하면 Outlook, 팀, Planner 등의 Microsoft 365 서비스에서 그룹을 만들 때 영향을 받게 됩니다.
  
지침: [Microsoft 365 그룹을 만들 수 있는 사용자 관리](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)
