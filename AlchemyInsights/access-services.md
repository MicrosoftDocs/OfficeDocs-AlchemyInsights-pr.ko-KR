# <a name="access-services"></a>Access Services

SharePoint Online에서 Access Services 기능 사용 중지 되 고 되었습니다. 새 Access 기반 웹 응용 프로그램 및 웹 데이터베이스에 더이상를 만들 수 및 년 4 월 2018 하 여 기존 웹 응용 프로그램 및 웹 데이터베이스를 종료 됩니다. 액세스 데스크톱 데이터베이스 (.accdb)이이 변경의 영향을 받지 않습니다. 액세스 하는 대신 Microsoft PowerApps 웹 응용 프로그램 및 웹 데이터베이스는 것이 좋습니다. 
  
[Microsoft PowerApps 하는 방법에 대 한 설명](https://powerapps.microsoft.com/)
