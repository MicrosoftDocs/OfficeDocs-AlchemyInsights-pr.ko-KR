# <a name="use-whiteboard-with-microsoft-365"></a>Microsoft 365와 함께 Whiteboard 사용하기

Microsoft Whiteboard는 사용자, 콘텐츠 및 아이디어가 함께 어우러지는 자유 형식의 디지털 캔버스입니다. 

- Microsoft 365 관리자로 [Whiteboard를 설정](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_07)하세요. 

- [Microsoft Teams와 함께 Whiteboard](https://support.microsoft.com/office/use-whiteboard-in-microsoft-teams-7a6e7218-e9dc-4ccc-89aa-b1a0bb9c31ee)를 사용하세요. 

- Whiteboard에서 다른 사용자와 [공동 작업](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_27)을 할 수 있습니다. 

자세한 내용은 [Microsoft Whiteboard 도움말](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920)을 참조하세요.
