# <a name="check-for-app-using-camera"></a>카메라를 사용하여 앱 확인

1. 시작을 **선택한** 다음 설정 > 개인 정보 보호 > 카메라(ms-settings:privacy-webcam)를 선택합니다. 
2. 응용 프로그램 목록에서 "현재 사용 중"인 앱을 찾아 응용 프로그램을 닫습니다.
