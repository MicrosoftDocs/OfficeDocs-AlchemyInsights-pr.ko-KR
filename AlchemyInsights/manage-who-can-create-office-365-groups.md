# <a name="manage-who-can-create-office-365-groups"></a>Office 365 그룹을 만들 수 있는 사용자 관리

기본적으로 조직의 모든 사용자가 O365 그룹을 만들 수 있습니다.
  
이를 변경 하려는 경우 사용자 하위 집합에 대 한 그룹 생성을 제한 하는 PowerShell을 사용 하 여 보안 그룹을 만들 수 있습니다.
  
단계별 지침을 따르십시오. [Office 365 그룹을 만들 수 있는 사용자 관리](https://support.office.com/article/4c46c8cb-17d0-44b5-9776-005fced8e618)
