# <a name="banned-sending-ip"></a>금지 보내기 IP

Microsoft 365로 메일을 보낼 때 보낸 사람에 게이 오류가 발생할 수 있습니다. 보낸 사람이 방문 https://sender.office.com 하 여 지침을 따라야 합니다.  포털에서 확인 전자 메일이 수신 되지 않으면 스팸 폴더를 확인 하 고 다른 전자 메일 주소를 사용 하 여 시도 합니다.
