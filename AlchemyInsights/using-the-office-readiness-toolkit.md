# <a name="using-the-office-readiness-toolkit"></a>Office 준비 도구 키트 사용

office 준비 도구 키트 사용에 대 한 자세한 내용은 [준비 도구 키트를 사용 하 여 office 365 ProPlus에 대 한 응용 프로그램 호환성 평가](https://docs.microsoft.com/DeployOffice/use-the-readiness-toolkit-to-assess-application-compatibility-for-office-365-pro)를 참조 하세요.

Readiness Toolkit에 대한 사용자 의견 또는 질문이 있는 경우 전자 메일(ReadyForOffice@microsoft.com)을 보내 주세요.
