# <a name="are-you-receiving-too-much-phish-or-spam-in-your-mailbox"></a>사서함에 피싱 또는 스팸이 너무 많이 있나요?

**참고** 에서 Microsoft 365 보안 센터에 액세스할 수 [https://security.microsoft.com](https://security.microsoft.com) 있습니다.

Microsoft 365 관리자는 피싱 또는 스팸 메시지가 배달된 이유를 확인하는 가장 좋은 방법은 제출 탐색기에서 Microsoft에 샘플을 [제출하는 것입니다.](https://security.microsoft.com/reportsubmission) 메시지 필터링과 관련된 즉각적인 피드백과 문제가 있는 이유를 얻을 수 있습니다.

사용자가 보고서 메시지 추가 기능을 통해 Microsoft에 직접 보고하도록 허용할 [수 있습니다.](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview)

**알고 있나요?**: 이제 메시지 추적에서 직접 샘플을 제출할 수 [있으며,](https://security.microsoft.com/messagetrace) E5/ATP 계획 2 고객은 위협 탐색기에서 샘플을 제출할 [수 있습니다.](https://docs.microsoft.com/microsoft-365/security/office-365-security/threat-explorer)
