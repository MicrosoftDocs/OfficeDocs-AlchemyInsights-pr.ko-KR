# <a name="do-you-have-legitimate-messages-being-marked-as-spam"></a>적법 한 메시지를 스팸으로 표시 되 고 있습니까?

위의 진단 시도 하거나이 [좋은 메시지를 스팸으로 표시 되 고 문제를 해결 하는 것에 대 한 단계별 가이드](https://docs.microsoft.com/en-us/office365/securitycompliance/prevent-email-from-being-marked-as-spam-0)를 사용 합니다.
  
이러한 유형의 문제를 보고할 가장 좋은 방법은입니다.
  
1. 사용자에 대 한 [보고 추가 기능을 활성화](https://docs.microsoft.com/en-us/office365/securitycompliance/enable-the-report-message-add-in?redirectSourcePath=%252fen-us%252farticle%252fEnable-the-Report-Message-add-in-4250c4bc-6102-420b-9e0a-a95064837676#entireorg) 합니다. 
    
2. 사용자 [메시지를 정크 메일 아님 보고서](https://support.office.com/en-us/article/use-the-report-message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2?ui=en-US&amp;rs=en-US&amp;ad=US)를 포함 합니다.
    
지원 티켓을 여는 것이 싶으면, 하십시오 연결에서 전체 전자 메일 머리글 적어도 **하나의 전체 원본 메시지** 아래 있습니다.
