# <a name="installation-hangs-or-gets-stuck"></a>설치가 중단 되거나 중지 됨



컴퓨터의 네트워크 연결, 바이러스 백신 소프트웨어 또는 Windows Update 설정으로 인해이 문제가 표시 될 수 있습니다. Here are some things you can try to fix the problem.
  
- 컴퓨터를 다시 시작 하 고 Office가 설치 되어 있는지 확인
    
- 유선 연결 사용
    
- 인터넷에 연결 되어 있지 않은 경우 [office offline installer를 사용 하](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA) 여 office 설치[](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA.aspx)
    
- 바이러스 백신 소프트웨어 일시적으로 끄기
    
- 제어판에서 온라인 복구 실행
    
- Office 제거 및 다시 설치
    
[Office를 설치 하는 데 오랜 시간이 걸리고 있습니다.](https://support.office.com/article/0f09f357-3fef-42a6-b8aa-cef4c6c44bdf?wt.mc_id=Alchemy_ClientDIA)
