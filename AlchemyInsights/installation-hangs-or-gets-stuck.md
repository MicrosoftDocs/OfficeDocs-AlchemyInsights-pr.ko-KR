# <a name="installation-hangs-or-gets-stuck"></a>설치 중단 되거나 문의할

사용자의 컴퓨터에서 네트워크 연결, 바이러스 백신 소프트웨어 또는 Windows Update 설정으로 인해이 동작을 볼 수 있습니다. 일부의 문제를 해결 하려면 시킬 수 있습니다.
  
- 컴퓨터를 다시 시작 하 고 확인 하는 경우에 Office 설치
    
- 유선된 연결을 사용 하 여
    
- 인터넷 연결 속도가 느린 경우 [Office 오프 라인 설치 관리자를 사용 하 여](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA.aspx) Office 설치
    
- 바이러스 백신 소프트웨어를 일시적으로 해제
    
- 제어판에서 온라인 복구를 실행 합니다.
    
- 제거 하 고 Office를 다시 설치
    
[Office는 시간이 오래 걸리는 설치 하려면](https://support.office.com/article/0f09f357-3fef-42a6-b8aa-cef4c6c44bdf?wt.mc_id=Alchemy_ClientDIA.aspx)
