# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>여러 전자 메일 주소에 대 한 착신 전환 설정

사용자 그룹에 대 한 전자 메일 전달을 설정 해야 하는 경우 배포 목록을 만들고 여러 명의 사용자를 추가한 다음 해당 목록으로 전달을 구성 합니다. 먼저 관리 센터에서 **그룹**  >  [그룹](https://portal.office.com/adminportal/home#/groups)으로 이동 합니다.
  
1. 그룹을 만듭니다.

2. 여러 사용자를 추가 합니다.

3. 해당 그룹에 대 한 사서함 전달을 구성 합니다.
