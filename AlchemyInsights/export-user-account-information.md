# <a name="export-user-account-information"></a>사용자 계정 정보 내보내기

라이선스, 암호 정책, 차단 된 계정, AAD 동기화 상태 등에 대 한 정보를 비롯 하 여 Microsoft 365 관리 센터에서 사용자 계정 정보를 내보내려면 **사용자** > [활성 사용자](https://go.microsoft.com/fwlink/p/?linkid=834822) 페이지로 이동 하 여 **사용자 내보내기를**선택 합니다.
