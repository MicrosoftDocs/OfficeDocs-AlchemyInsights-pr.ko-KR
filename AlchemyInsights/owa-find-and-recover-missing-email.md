# <a name="find-and-recover-missing-or-deleted-email-calendar-events-or-contacts"></a>누락 되었거나 삭제 된 전자 메일, 일정 이벤트 또는 연락처 찾기 및 복구

- 메일에서 페이지 위쪽의 **검색** 창을 사용 하 여 [메시지나 연락처를 검색](https://support.office.com/article/b27e5eb7-3255-4c61-bf16-1c6a16bc2e6b)합니다.

- 일정에서 페이지 위쪽의 **검색** 창을 사용 하 여 [단어, 이름 또는 구를 사용 하 여 이벤트를 검색](https://support.office.com/article/d587aaec-fb2c-4f6f-aee1-0df1fc591477)합니다. 달력을 두 개 이상 열어 둔 경우 현재 선택한 달력에 포커스가 검색 됩니다.

- 실수로 메시지, 일정 이벤트 또는 연락처를 삭제 한 경우 [삭제 된 항목 또는 정크 메일 폴더에 있는 항목은 복구할](https://support.office.com/article/a8ca78ac-4721-4066-95dd-571842e9fb11)수 있습니다.
