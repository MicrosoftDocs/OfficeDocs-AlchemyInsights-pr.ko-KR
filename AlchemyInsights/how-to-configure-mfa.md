# <a name="configure-multi-factor-authentication-in-office-365"></a>Office 365에서 다단계 인증 구성

MFA를 구성 하려면 [Office 365 사용자를 위한 다단계 인증 설정을](https://docs.microsoft.com/office365/admin/security-and-compliance/set-up-multi-factor-authentication)참조 하세요.

MFA를 사용 하도록 설정 하 고 구성 하려면 Office 365 전역 관리자 여야 합니다.
