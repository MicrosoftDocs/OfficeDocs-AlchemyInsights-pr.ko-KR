# <a name="update-payment-details"></a>결제 정보 업데이트

1. 관리 센터에서 [결제 방법](https://go.microsoft.com/fwlink/p/?linkid=2018806) 페이지로 이동 하거나 **대금 청구** \> **지불 방법을**선택 합니다.
    
    > [!NOTE]
    > 업데이트가 완료 되기 전에 관리자 홈 페이지로 다시 전송 하는 루프가 발생 하면 Inprivate 또는 Incognito 모드를 시도해 봅니다. 
  
2. 업데이트할 신용 카드를 선택 합니다.
    
    > [!NOTE]
    > 다른 관리자가 결제 방법을 추가 하면 신용 카드 또는 은행 계좌 정보를 업데이트할 수 없게 됩니다. 신용 카드가 만료 되 고 다른 관리자가이를 업데이트할 수 없는 경우 **결제 방법 추가**를 선택 하 여 다른 신용 카드나 새 은행 계좌를 추가 하 여 서비스 중단을 방지할 수 있습니다. 
  
3. **결제 정보** 페이지에서 **결제 정보 편집**을 선택 합니다.
    
4. 업데이트를 만든 다음 **저장**을 선택 합니다.
    
5. 이 결제 방법을 제거 하려면 **삭제**를 선택 합니다.
    
자세한 내용은 [비즈니스용 Office 365에서 신용 카드 또는 은행 계좌 추가, 업데이트 또는 제거](https://support.office.com/article/30ba9c83-50d8-4020-90ed-830a5b8c8724) 및 [신용 카드 및 은행 계좌 문제 해결](https://support.office.com/article/30ba9c83-50d8-4020-90ed-830a5b8c8724)을 참조 하세요.
