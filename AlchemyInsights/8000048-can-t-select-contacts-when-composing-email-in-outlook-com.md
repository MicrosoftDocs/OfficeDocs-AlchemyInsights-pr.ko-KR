# <a name="cant-select-contacts-when-composing-email"></a>전자 메일 메시지를 작성할 때 대화 상대를 선택할 수 없습니다.

때 일부 고객 새 전자 메일을 작성 하 고 **받는** 사람 또는 **참조** 단추를 클릭 하를 자신의 대화 상대 목록이 표시 되지 않을 수 있습니다. 이름을 선택 하려면 **받는** 사람 또는 **참조** 필드에 입력 하기 시작 하 고 관련된 대화 상대가 표시 됩니다.
