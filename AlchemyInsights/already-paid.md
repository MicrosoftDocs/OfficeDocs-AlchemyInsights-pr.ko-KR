# <a name="already-paid"></a>이미 지불?

지불 이미 보낸 수 보겠습니다 알고 있는 경우:
  
1. [구독](https://go.microsoft.com/fwlink/p/?linkid=842054) 페이지로 이동 하거나 **대금 청구** 선택 \> **구독**합니다.
    
2. 구독을 선택 하 고 **내 bill** 섹션에서 **보기 Bill**를 선택 합니다.
    
3. **Bill 세부 정보** 페이지에서 선택 **이미 지불?** 에 대 한 발송 된 지불와 관련 된 정보를 입력 하 라는 메시지가 표시 됩니다. 
    
4. 필요한 모든 정보를 입력 하는 경우에 **보내기**를 선택 합니다.
