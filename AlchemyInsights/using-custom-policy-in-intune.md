# <a name="using-custom-policy-in-intune"></a>Intune에서 사용자 지정 정책 사용

Intune에는 Android, iOS, macOS 및 Windows 장치용으로 제공 되는 다양 한 설정이 있습니다. 사용자 지정 프로필을 만든 다음 Intune을 사용 하 여 이러한 사용자 지정 설정을 장치에 배포할 수도 있습니다. 사용자 지정 프로필은 Intune에 기본적으로 제공 되지 않는 장치 설정 및 기능을 추가 하도록 설계 되었습니다.

[Intune의 Android 장치에 대 한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-android)

[Intune의 Android Enterprise 장치에 대 한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Intune의 iOS 장치에 대 한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-ios)

[Intune의 macOS 장치에 대 한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-macos)

[Intune의 Windows 10 장치에 대 한 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Intune의 Windows Holographic for Business 장치용 사용자 지정 설정](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
