# <a name="what-is-autosave"></a>자동 저장 이란?

자동 저장은 작업을 수행 하는 동안 몇 초 마다 파일을 자동으로 저장 하는 Excel, Word 및 PowerPoint for Office 365 구독자에서 사용할 수 있는 기능입니다. 

자주 묻는 질문 [은 자동 저장 이란?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5) 를 참조 하세요.
