# <a name="enable-seamless-sso"></a>원활한 SSO 사용

Azure Active Directory (Azure AD) 원활한 SSO (고급형 Single Sign-on)가 회사 네트워크에 연결 된 회사 데스크톱에 있는 경우 사용자에 게 자동으로 로그인 합니다.
  
[원활한 SSO를 사용 하도록 설정 하려면이 문서의 단계를 따르세요.](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
