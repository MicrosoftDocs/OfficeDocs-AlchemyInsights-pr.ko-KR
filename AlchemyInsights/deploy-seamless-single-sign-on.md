# <a name="enable-seamless-sso"></a>원활 하 게 SSO를 사용 하도록 설정

Azure Active Directory (Azure AD) 원활 하 게 Single Sign-on (원활 하 게 SSO) 자동으로 로그인 사용자가 회사 네트워크에 연결 된 회사 데스크톱에 있을 때 합니다.
  
[원활 하 게 SSO를 사용 하도록 설정 하려면이 문서의 단계를 수행 합니다.](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
