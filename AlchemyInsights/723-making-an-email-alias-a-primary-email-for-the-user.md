# <a name="make-an-email-alias-the-primary-address-for-a-user"></a>사용자에 대 한 기본 주소는 전자 메일 별칭 확인

자세한 내용은 [사용자에 대 한 다른 전자 메일 별칭 추가](https://support.office.com/article/0b0bd900-68b1-4bf5-808b-5d240a7739f4)참조 하십시오.
  
 **참고**: 온-프레미스 Exchange 조직에서 기본 전자 메일 주소를 변경 하려면 필요한 조직 하이브리드 모드인 경우. 자세한 내용은 [사서함에 대 한 추가 또는 제거 전자 메일 주소](https://technet.microsoft.com/library/bb123794.aspx)를 참조 하십시오.
