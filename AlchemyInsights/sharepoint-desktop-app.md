# <a name="desktop-app-for-sharepoint"></a>SharePoint 용 데스크톱 앱

조직이 SharePoint 용 데스크톱 앱에 대 한 개인 미리 보기 프로그램에 입력 됩니다.

현재 Microsoft는 SharePoint 용 데스크톱 앱 (비공개 미리 보기)을 지원 하지 않습니다. SharePoint 용 데스크톱 앱에 대 한 지원을 받으려면 참여 한 Microsoft 비공개 미리 보기 파트너를 사용 하세요.
