# <a name="sharepoint-online-throttling"></a>SharePoint Online 제한

SharePoint Online은 제한된 기능을 사용하여 SharePoint Online 서비스의 최상의 성능과 안정성을 유지합니다. 제한 기능은 사용자 작업 수 또는 동시 호출 수(스크립트 또는 코드에 의한)를 제한하여 리소스가 과도하게 사용되지 않도록 합니다. 제한 된 시간이 되 면 사용자 지정 코드로 인해 발생 한 99%가 됩니다.
  
제한에 대 한 자세한 내용은 [SharePoint Online에서 제한 또는 차단 방지](https://go.microsoft.com/fwlink/?linkid=2022019) 를 참조 하세요.
