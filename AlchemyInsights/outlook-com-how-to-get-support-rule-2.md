# <a name="how-to-get-support-in-outlookcom"></a>Outlook.com에서 지원을 받는 방법

죄송 하지만 Outlook.com에서는 전화 지원을 제공 하지 않지만 도움이 될 수 있는 전용 팀이 있습니다.
질문을 입력 하거나 위의 **도움말** 상자에 문제를 설명 하 고 **도움말 보기**를 선택 합니다.
