# <a name="how-to-purchase-education-offer"></a>교육 서비스를 구매 하는 방법

Office 365 Education을 구입할 수 있도록 승인받았습니다. 구입하려면 다음과 같이 하세요.
  
1. 관리 센터에서 **Billing** \> **[제품](https://go.microsoft.com/fwlink/p/?linkid=842054)** 청구 페이지로 이동 합니다.

2. **학생 또는 교직원**에 대 한 평가판을 선택 합니다.

3. **라이선스** 섹션에서 **구독 구매**를 선택 합니다.

4. 나머지 단계를 수행 하 여 체크 아웃 합니다.

평가판이 아닌 다른 구독을 구매 하려면 **결제** \> [구매 서비스](https://go.microsoft.com/fwlink/p/?linkid=868433) 페이지로 이동 하 여 **교육 계획**아래의 목록에서 선택 합니다.

자세한 내용은 [평가판과 동일한 요금제 구매](https://docs.microsoft.com/microsoft-365/commerce/subscriptions-and-billing/buy-a-subscription-from-your-free-trial#buy-the-same-plan-as-your-trial)를 참조 하세요.
