# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>웹용 Outlook에서 공유 사서함 열기 및 사용

사용자가 액세스할 수 있는 공유 사서함은 웹 (이전의 outlook web App)에서 outlook에 자동으로 표시 되지 않습니다.

공유 사서함과 기본 사서함을 함께 표시 하거나 공유 사서함을 별도의 브라우저 창에서 열 수 있습니다. 공유 사서함에 대 한 새 메시지 알림을 받으려면 공유 사서함을 별도의 브라우저 창에서 열어야 합니다. 자세한 내용은 [Outlook Web App에서 공유 사서함 열기 및 사용](https://support.office.com/article/BC127866-42BE-4DE7-92AE-1EF2F787FD5C)을 참조 하세요.
