# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>SaRA를 사용 하 여 문제를 진단 하 고 Outlook 인증 문제를 해결 하려면

사용 하 여는 [Outlook에서 계속 내 암호에 대 한 요청](https://aka.ms/SaRA-OutlookPwdPrompt) 진단 영향을 받는 컴퓨터 문제를 해결 Outlook이 계속 해 서 암호를 표시 하는 것이 좋습니다. 이 [SaRA](https://diagnostics.office.com/#/) 진단 자동 검사를 수행 하 고 발견 된 문제를 해결 하는 사용할 수 있는 가능한 솔루션을 반환 합니다.
