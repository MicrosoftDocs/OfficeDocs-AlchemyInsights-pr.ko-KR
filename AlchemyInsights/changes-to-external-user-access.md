# <a name="changes-to-external-user-access"></a>외부 사용자 액세스에 대 한 변경 내용

2018 년 3 월 23 일부 부터는 외부 사용자가 모든 사용자, 인증 된 사용자 또는 모든 양식 사용자와 공유 된 콘텐츠에 더 이상 액세스할 수 없습니다. 외부 사용자는 사용자와 공유 된 콘텐츠나 사용자가 속한 그룹에만 액세스할 수 있습니다. 기존 설정을 유지 하려는 경우 다음을 참조 하세요.https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
