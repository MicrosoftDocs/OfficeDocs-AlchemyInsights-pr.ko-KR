# <a name="organization-policy-settings-for-external-users"></a>외부 사용자에 대 한 조직 정책 설정

조직의 외부 사용자와 공유 하려고 하면 사용자에 게 다음과 같은 경고 메시지가 표시 될 수 있습니다. 

   **조직의 정책에서는 이러한 사용자와의 공유를 허용 하지 않습니다. Microsoft 365 관리 센터에서 외부 공유로 이동 하 여이 기능을 사용 하도록 설정 합니다.** 

이 문제는 일반적으로 외부 사용자와의 공유를 시도 하거나 새 외부 사용자와 공유할 때 발생 합니다. 자세한 내용은 다음 문서를 참조 하세요.

[조직의 정책에 따라 이러한 사용자와 공유할 수 없습니다.](https://docs.microsoft.com/sharepoint/support/administration/organization-policies-do-not-allow-you-to-share-with-users-error)
