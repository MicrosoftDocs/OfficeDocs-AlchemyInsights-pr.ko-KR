[<span data-ttu-id="992ce-103">비디오: 손상 된 Office 365 계정 수정</span><span class="sxs-lookup"><span data-stu-id="992ce-103">VIDEO: Fixing a compromised Office 365 account</span></span>](https://www.microsoft.com/videoplayer/embed/RE2jvOb?pid=ocpVideo0-innerdiv-oneplayer&amp;postJsllMsg=true&amp;maskLevel=20&amp;autoplay=true)
  
1. <span data-ttu-id="992ce-p101">[사용자 암호를 즉시 다시 설정](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) 합니다. 최종 사용자에 게 전자 메일을 통해 새 암호를 전달 하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="992ce-p101">[Reset the user's password](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) immediately. Do not communicate the new password through email to the end user.</span></span> 
    
2. <span data-ttu-id="992ce-106">사서함 수준에서 설정 된 의심 스러운 [전달 주소](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) 를 제거 합니다.</span><span class="sxs-lookup"><span data-stu-id="992ce-106">Remove any suspicious [forwarding addresses](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) set at the mailbox level.</span></span> 
    
3. <span data-ttu-id="992ce-107">사서함 내에서 의심 스러운 [받은 편지함 규칙](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) 집합을 제거 합니다.</span><span class="sxs-lookup"><span data-stu-id="992ce-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span> 
    
4. <span data-ttu-id="992ce-p102">사용자가 전자 메일을 보낼 수 없도록 차단 된 경우 [제한 된 사용자로 이동 하 여 계정을 차단 해제](https://protection.office.com/?hash=/restrictedusers)합니다. 작업이 완료 되 면 사용자는 1 시간 내에 메시지 보내기를 다시 시작할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="992ce-p102">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers). Once done, the user should be able to resume sending messages within 1 hour.</span></span>
    
5. <span data-ttu-id="992ce-110">계정이 더 이상 손상 되지 않음을 확신할 때까지 [관리 역할 그룹](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) 에서 사용자 계정을 제거 합니다.</span><span class="sxs-lookup"><span data-stu-id="992ce-110">Remove the user account from any [administrative role groups](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) until you are confident that the account is no longer compromised.</span></span> 
    
<span data-ttu-id="992ce-111">나중에 데이터 위반 또는 손상 된 계정이 발생할 가능성을 최소화 하려면 [Office 365 보안 모범 사례 문서](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3)를 읽는 것이 좋습니다.</span><span class="sxs-lookup"><span data-stu-id="992ce-111">To minimize the potential of a data breach or a compromised account in the future, we recommend reading our [Office 365 Security best practices article](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).</span></span>
