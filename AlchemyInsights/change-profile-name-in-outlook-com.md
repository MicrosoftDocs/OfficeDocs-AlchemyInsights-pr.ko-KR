# <a name="change-your-profile-name"></a>프로필 이름 변경

1. [Microsoft 계정 정보](https://go.microsoft.com/fwlink/p/?linkid=860841)로 이동 합니다.
    
2. **이름 편집**을 선택 합니다. 
    
3. 이름 및 성을 입력 하 고 **저장**을 선택 합니다. 
    
프로필 이름 변경 내용을 적용 하려면 24 시간까지 허용 하세요.
  
자세한 내용은 [Outlook.com에서 프로필 사진 또는 이름 변경을](https://go.microsoft.com/fwlink/?linkid=873110)참조 하세요.
