# <a name="help-understanding-value-added-tax-vat"></a>VAT(부가가치세) 이해 지원

영국과 같이 VAT(부가가치세)를 사용하는 국가에 있는 경우 Microsoft Invoicing은 VAT를 계산하여 송장에 금액을 적용합니다.
  
세금을 이유로 Invoicing 설정에서 VAT 등록 번호도 지정해야 합니다.
  
VAT 요금을 추가하거나 제거할 수 없지만 설정에서 기본 VAT 비율을 변경할 수 있습니다. 기본적으로 Invoicing은 서비스를 사용하여 VAT 등록 번호의 유효성을 검사하지만 유럽 위원회에서 게시한 [VIES](https://go.microsoft.com/fwlink/?LinkID=841741) 고지 조항을 읽어보는 것이 좋습니다. 서비스를 사용하지 않도록 설정하는 경우 모든 VAT 등록 번호를 확인해야 합니다.
