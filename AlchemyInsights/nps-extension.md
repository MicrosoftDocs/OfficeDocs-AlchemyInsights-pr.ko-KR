# <a name="network-policy-server-extension"></a>네트워크 정책 서버 확장

Azure AD 다단계 인증용 NPS(네트워크 정책 서버) 확장은 기존 서버를 사용하여 인증 인프라에 클라우드 기반 MFA 기능을 추가합니다. NPS 내선 번호를 사용하면 새 서버를 설치, 구성 및 유지 관리하지 않고도 전화 통화, 문자 메시지 또는 전화 앱 확인을 기존 인증 흐름에 추가할 수 있습니다.

NPS 확장은 RADIUS와 클라우드 기반 Azure AD 다단계 인증 간의 어댑터 역할을 하여 페더러 또는 동기화된 사용자에 대해 두 번째 인증 단계 인증을 제공합니다.

NPS 확장의 작동 방식에 대한 자세한 내용은 [Azure AD Multi-Factor Authentication과 기존 NPS(네트워크](https://docs.microsoft.com/azure/active-directory/authentication/howto-mfa-nps-extension)정책 서버) 인프라 통합을 참조하세요.
