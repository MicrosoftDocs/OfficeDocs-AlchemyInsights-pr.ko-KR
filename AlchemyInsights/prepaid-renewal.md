# <a name="prepaid-renewal"></a>선불 갱신

1. 관리 센터에서 **청구** \> [구독](https://go.microsoft.com/fwlink/p/?linkid=842054) 페이지로 이동 합니다.

2. 연장할 선불 구독을 선택 하 고 **확장**을 선택 합니다.

3. **제품 키**에 대 한 옵션을 선택 합니다. (이 제품은 소매점에서 구입한 제품 키 인 경우 다른 결제 방법으로 확장 하는 옵션을 사용할 수 있습니다.)

4. 제품 키를 입력 합니다.

5. 정보가 올바른지 확인 하 고 **다음**을 선택 합니다.

6. 라디오 단추를 선택 하 여 기존 라이선스의 종료 날짜를 확장 하 고 **다음**을 선택 합니다.

새 관리 센터를 사용 하 고 있나요? 새 단계는 [제품 키를 사용 하 여 유료 구독 추가 또는 확장](https://docs.microsoft.com/en-us/office365/admin/misc/add-licenses-using-product-key)을 참조 하세요.
