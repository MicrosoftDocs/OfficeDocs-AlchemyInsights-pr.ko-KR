# <a name="prepaid-renewal"></a>선불된 갱신

1. 관리 센터에서 [구독](https://go.microsoft.com/fwlink/p/?linkid=842054) 페이지로 이동하거나 **청구** \> **구독**을 선택합니다.
    
2. 를 확장 하 고 **확장**을 선택 하려면 선불된 구독을 선택 합니다.
    
3. **제품 키**에 대 한 옵션을 선택 합니다. (소매 위치에서 구입한 제품 키가 할 경우 다른 지불 방법을 통해 확장 하는 옵션입니다.)
    
4. 제품 키를 입력 합니다.
    
5. 정보가 올바른지 확인 한 다음 **다음**을 선택 합니다.
    
6. 기존 라이선스의 종료 날짜를 확장 하 고 **다음**을 선택 하 고 라디오 단추를 선택 합니다.
