# <a name="delete-a-folder"></a>폴더 삭제

1. 왼쪽 창에서 삭제할 폴더 또는 하위 폴더를 마우스 오른쪽 단추로 클릭 하 고 **폴더 삭제**를 선택 합니다.
2. **확인** 을 선택 합니다.

자세한 내용은 [Outlook.com에서 폴더 삭제](https://go.microsoft.com/fwlink/p/?linkid=873134)를 참조 하십시오.
