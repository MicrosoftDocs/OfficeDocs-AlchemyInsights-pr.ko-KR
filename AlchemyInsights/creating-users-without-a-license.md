# <a name="create-mailboxes-that-dont-require-licenses"></a>라이선스가 필요 없는 사서함 만들기

- 여러 사용자가 그룹 전자 메일 주소를 필요로 하는 경우에 [는 공유 사서함 만들기](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox)를 참조 하세요.

- 팀이 함께 작업 하 고 전자 메일 및 파일을 공유할 수 있도록 하려면 [Microsoft 365 그룹에 대 한](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2)자세한 정보를 참조 하세요.
