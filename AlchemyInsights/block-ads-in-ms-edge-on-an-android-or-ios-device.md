# <a name="block-ads-in-microsoft-edge-on-an-adroid-or-ios-device"></a>Adroid 또는 iOS 장치에서 Microsoft Edge에서 광고 차단

AdBlock Plus에서 허용되는 것으로 인정되는 광고를 제외한 모든 광고를 차단하는 방법:
1. Select **...** > **설정**  >  **콘텐츠 차단 .**
2. 광고를 차단합니다.

모든 광고를 차단하는 방법:
1. Select **...** > **설정**  >  **콘텐츠 차단 .**
2. 광고를 차단합니다.
3. 고급 **설정을 선택합니다.**
4. 허용 가능한 광고 허용을 끄기.
