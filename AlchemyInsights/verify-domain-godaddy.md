# <a name="verify-your-domain-with-godaddy"></a>GoDaddy와 도메인 확인

도메인을 확인 하려면 도메인 등록 기관에 TXT 레코드를 추가 하 여 도메인을 소유 하 고 있는지 확인 해야 합니다. 

먼저 [Add Domain wizard](https://admin.microsoft.com/Adminportal#/Domains) **설정** \> **도메인**을 선택한 다음 **도메인 추가**마법사를 실행 합니다.
  
단계별 지침은 [GoDaddy에서 DNS 레코드 만들기 365를](https://docs.microsoft.com/microsoft-365/admin/dns/create-dns-records-at-godaddy) 참조 하세요.
