# <a name="create-sharepoint-sites-using-templates"></a>템플릿을 사용 하 여 SharePoint 사이트 만들기

SharePoint 사이트 서식 파일은 특정 비즈니스 요구를 중심으로 디자인 된 미리 만들어진 정의입니다. 자세한 내용은 <a href="https://support.office.com/en-us/article/using-templates-to-create-different-kinds-of-sharepoint-sites-449eccec-ff99-4cf3-b62e-dcfee37e8da4">서식 파일을 사용 하 여 여러 종류의 SharePoint 사이트 만들기</a> 를 참조 하세요.</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">다음은 Sharepoint Online에서 사이트 또는 목록을 서식 파일로 저장 하는 방법에 대 한 몇 가지 일반적인 문제/해결 방법입니다.</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">사이트 저장/목록 서식 파일 단추가 없거나 사용할 수 없습니다.</span></u></strong></p> <ul> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">관리자는 서식 파일 기능을 사용 하도록 설정 하기 위해 사용자 지정 스크립트를 허용 해야 합니다. 자세한 단계 </span> </span>는 <a style="orphans: 2; -webkit-text-stroke-width: 0px; word-spacing: 0px;" href="https://docs.microsoft.com/en-us/sharepoint/allow-or-prevent-custom-script">허용 또는 금지 사용자 지정 스크립트</a>를 참조 하십시오.</li> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">사이트를 서식 파일로 저장 명령은 지원 되지 않으며 SharePoint Server 게시 인프라를 사용 하는 사이트에서 문제를 발생 시킬 수 있습니다.</span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">사이트 서식 파일을 만들 수 없거나 제대로 작동 하지 않음</span></u></strong></p> <ul> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">서식 파일에 <a href="https://social.technet.microsoft.com/wiki/contents/articles/14423.sharepoint-2013-existing-features-guid.aspx">기능이</a> 누락 되 고 활성화가&rsquo;성공 했을 수 있습니다. 현재 사이트 모음에서 정품 인증 기능을 사용할 수 없는 경우에는 사이트 서식 파일을 사용 하 여 사이트를 만들 수 없습니다.</span></li> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">목록 또는 라이브러리가 사이트 서식 파일의 생성을 차단할 수 있는 것으로 표시 되는 것으로 5000 항목 <a href="https://support.office.com/en-us/article/Manage-large-lists-and-libraries-in-SharePoint-B8588DAE-9387-48C2-9248-C24122F07C59">의 목록이</a> 나 라이브러리를 초과 하는지 확인 합니다.</span></li> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">사이트에서 너무 많은 리소스를 사용 하 고 있기 때문에 사이트 서식 파일이 50 MB 제한을 초과 합니다.</span></li> <li>
조회 열을 사용 하는 목록의 데이터를 표시 하는 데 문제가 있습니다. 자세한 내용은 Template 생성 목록 </span> <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> <a style="box-sizing: border-box; -webkit-text-stroke-width: 0px; word-spacing: 0px;" href="https://support.office.com/en-us/article/template-generated-list-doesn-t-display-correct-data-for-a-column-in-sharepoint-online-20430b62-e40c-4f6f-8889-aa24e80d605a"> <span style="color: #0067b8; text-decoration: none; text-underline: none;">&rsquo;에서 SharePoint Online의 올바른 조회 목록의 데이터를 표시 하지 않습니다 .를 참조 하세요.
