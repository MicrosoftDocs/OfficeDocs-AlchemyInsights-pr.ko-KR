# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>새로 고침 시나리오 문제 해결-타일 오류

Power BI 서비스 내에서 타일 새로 고침 오류가 발생 하는 경우 새로 고침이 대시보드 타일에 반영 되도록 약 10-15 분 정도 기다립니다. 여전히 표시 되지 않으면 시각화를 대시보드에 다시 고정 합니다.

자세한 내용:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
