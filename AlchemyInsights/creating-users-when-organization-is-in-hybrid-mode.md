# <a name="create-users-in-hybrid-deployments"></a>하이브리드 배포에서 사용자 만들기

조직이 온-프레미스 환경과 Microsoft 365 간의 하이브리드 배포 인 경우 온-프레미스 Active Directory에서 모든 사용자 개체를 만들어야 합니다. 조직에 동기화 되도록 해당 OU (조직 구성 단위)에 사용자 개체를 만듭니다.
