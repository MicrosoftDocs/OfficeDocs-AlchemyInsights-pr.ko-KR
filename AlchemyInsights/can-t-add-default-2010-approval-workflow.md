# <a name="cant-add-default-2010-approval-workflow"></a>기본 2010 승인 워크플로를 추가할 수 없음

Microsoft SharePoint 사이트 모음에서는 목록 또는 라이브러리에 전역으로 다시 사용 가능한 워크플로 (예: "승인-SharePoint 2010")를 추가할 수 없습니다. 승인 워크플로가 누락 되었을 수 있습니다. 이 문제를 해결 하려면 [SharePoint 목록 또는 라이브러리에 전역으로 다시 사용 가능한 워크플로를 추가할 수 없음을](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library)참조 하세요.

워크플로 승인에 대 한 자세한 내용은 [모든 승인 워크플로 정보] (https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742)를 참조 하세요. 
 
자세한 내용은 [최신 승인 흐름] (https://flow.microsoft.com/blog/introducing-modern-approvals)을 참고 하세요.
