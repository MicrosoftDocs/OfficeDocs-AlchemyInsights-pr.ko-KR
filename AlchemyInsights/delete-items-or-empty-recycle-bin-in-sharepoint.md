# <a name="delete-items-or-empty-the-recycle-bin-of-a-sharepoint-site"></a>SharePoint 사이트의 휴지통 항목 삭제 또는 비우기 

SharePoint 문서 라이브러리 또는 목록에서 항목을 삭제 하면 SharePoint에서 즉시 제거 되지 않습니다. 삭제 된 항목은 일정 기간 동안 사이트 휴지통으로 이동 합니다. 이 시간 동안 삭제 한 항목을 원래 위치로 복원 하거나 제거 하 여 저장 공간을 확보할 수 있습니다.
