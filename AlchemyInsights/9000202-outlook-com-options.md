# <a name="how-to-find-options-in-outlookcom-settings"></a>Outlook.com 설정에서 옵션을 찾는 방법

페이지 맨 위에서 톱니 바퀴 아이콘을 선택 하 여 설정을 엽니다. 설정 창 아래쪽에서 **모든 Outlook 설정 보기**를 선택 합니다.

이렇게 하면 일반, 전자 메일, 일정 및 해당 하는 경우 프리미엄 설정 및 옵션을 볼 수 있습니다.
