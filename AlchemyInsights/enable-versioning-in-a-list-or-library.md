# <a name="enable-versioning-for-a-list-or-library"></a>목록 또는 라이브러리에 대 한 버전 관리 사용

1. 목록 또는 라이브러리로 이동 합니다.
    
2. **설정을**클릭 하 고 **라이브러리 설정** 또는 **목록 설정을**클릭 합니다.
    
3. **버전 관리 설정을**클릭 합니다.
    
4. 원하는 설정을 선택 하 고 **확인**을 클릭 합니다.
