# <a name="create-a-poll-form-or-quiz-for-teams-with-microsoft-forms"></a>Microsoft Forms로 Teams에 대한 설문 조사, 양식 또는 퀴즈 만들기

Microsoft Teams에서 양식, 설문 조사 또는 퀴즈를 만들려고 하나요? Microsoft Forms가 도울 수 있습니다!

 - 조직에 대한 [Forms를 설정하거나 해제](https://support.office.com/article/turn-off-or-turn-on-microsoft-forms-8dcbf3ab-f2d6-459a-b8be-8d9892132a43)하거나 특정 사용자에 대해서만 사용하도록 설정합니다.
 
 - Microsoft Teams에서 양식 [만들기, 편집 및 공동 작업](https://support.office.com/article/create-edit-and-collaborate-on-a-form-in-microsoft-teams-333b97a3-41d9-48bc-a1cb-84a96bd44e14)합니다.
 
 - Microsoft Forms에 대한 [질문과 대답](https://support.office.com/article/get-started-1dd58027-40dc-42d0-9ca4-80ddecc5c696)
