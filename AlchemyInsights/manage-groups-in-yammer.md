# <a name="manage-groups-in-yammer"></a>Yammer에서 그룹 관리

- **그룹 만들기 권한** Yammer는 그룹을 만들 수 있는 사용자를 제어 하지 않습니다. 그러나 연결 된 Microsoft 365 그룹에 대 한 그룹 만들기를 관리할 수 있습니다. 자세한 내용은 [Microsoft 365 그룹을 만들 수 있는 사용자 관리](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups) 를 참조 하세요.

- **동적 그룹 구성원** Yammer [에서 동적 그룹 만들기](https://docs.microsoft.com/yammer/manage-yammer-groups/create-a-dynamic-group) 에 대 한 자세한 내용은 yammer에 연결 된 Microsoft 365 Groups를 참조 하세요.

- **삭제 된 그룹 복원** Yammer 그룹 복원 가능 삭제 된 [Microsoft 365 group을](https://docs.microsoft.com/microsoft-365/admin/create-groups/restore-deleted-group) 통해 가능한 그룹 연결 되지 않은 그룹은 삭제 후 복원할 수 없습니다.

그룹을 관리 하는 방법에 대 한 자세한 내용은 [Yammer에서 그룹 관리](https://support.office.com/article/Manage-a-group-in-Yammer-6e05c6d6-5548-4c88-89cd-e6757a514ef2)를 참조 하십시오. 연결 된 Microsoft 365 그룹에 대 한 자세한 내용은 [Yammer 및 Microsoft 365 groups](https://docs.microsoft.com/yammer/manage-yammer-groups/yammer-and-office-365-groups)를 참조 하십시오.
