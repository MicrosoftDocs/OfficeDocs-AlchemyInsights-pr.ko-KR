# <a name="create-a-shared-mailbox"></a>공유 사서함 만들기

공유 사서함에 둘 이상의 사용자를 다른 라이선스를 할당 하지 않고 sales@contoso.com, 같은 별칭에 대 한 전자 메일을 주고받을 수 있습니다. 공유 사서함을 만들려면 [공유 전자 메일 가이드](https://portal.office.com/adminportal/home)로 이동 합니다.
  
팁 및 공유 사서함을 사용 하는 방법에 대 한 추가 정보에 대 한 [공유 사서함 만들기](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) 문서를 참조 하십시오.
