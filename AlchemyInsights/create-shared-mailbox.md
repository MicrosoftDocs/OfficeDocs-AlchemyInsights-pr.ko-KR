# <a name="create-a-shared-mailbox"></a>공유 사서함 만들기

공유 사서함을 사용 하면 다른 라이선스를 할당 하지 않고 sales@contoso.com 같은 별칭에 대 한 전자 메일을 보내고 받을 수 있습니다. 

공유 사서함 [만들기](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) 문서를 참조 하 여 팁에 대 한 추가 정보를 확인 하세요.
