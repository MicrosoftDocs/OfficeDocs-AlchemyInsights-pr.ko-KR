# <a name="error-onedrive-cannot-connect-to-windows"></a>오류: OneDrive에서 Windows에 연결할 수 없음

오류 메시지를 해결 하려면 cldflt.sys 파일을 검색 하 여 실행 중인지 확인 하 고, 그렇지 않은 경우 사용 하도록 설정 합니다. 
  
cldflt.sys 파일을 사용 하도록 설정 하는 방법에 대 한 자세한 내용은 [onedrive의 파일에 액세스할 때 onedrive에서 Windows에 연결할 수 없음 "오류](https://go.microsoft.com/fwlink/?Linkid=2031032) 를 참조 하세요.
