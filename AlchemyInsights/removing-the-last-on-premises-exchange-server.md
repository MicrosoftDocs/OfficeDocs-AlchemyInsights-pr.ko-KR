# <a name="removing-the-last-on-premises-exchange-server"></a>마지막 온-프레미스 Exchange 서버를 제거합니다.

온-프레미스 조직에서 마지막 Exchange 서버를 제거 하기 전에 다른 목적을 위해 해당 마지막 서버 해야 하는 경우를 결정 하려면이 작업의 의미를 이해 해야 합니다. 자세한 내용은 체크 아웃 [방법 및 하이브리드 배포에서 온-프레미스 Exchange 서버를 해제 하는 경우](https://technet.microsoft.com/en-us/library/dn931280%28v=exchg.150%29.aspx)합니다.
