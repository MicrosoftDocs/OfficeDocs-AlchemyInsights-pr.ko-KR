# <a name="turn-external-sharing-on-or-off-for-a-sharepoint-site"></a>SharePoint 사이트에 대해 외부 공유 설정 또는 해제

클래식 사이트 모음의 경우 다음을 수행 합니다.
  
1. 기본 SharePoint 관리 센터의 왼쪽 창에서 **사이트 모음**을 클릭 합니다.
    
2. 사이트 또는 사이트를 선택 하 고 리본 메뉴에서 **공유**를 클릭 합니다.
    
기본적으로 사용자가 만드는 새 사이트 (예를 들어, Office 365 그룹 및 통신 사이트에 속하는 팀 사이트)는 조직 전체 설정과 동일한 공유 설정을 사용 합니다. 로그인 해야 합니다. 이 경우 사이트는 로그인 하는 신규 및 기존 외부 사용자와의 공유를 허용 합니다. 특정 사이트에 대 한 설정을 변경 하려면 새 SharePoint 관리 센터 (미리 보기)를 사용 합니다.
  
> [!NOTE]
> 모든 사이트에 대 한 외부 공유 설정은 조직 차원의 설정 보다 더 제한적으로 사용할 수 있지만 조직 전체 설정 보다는 허용이 더 이상 가능 하지 않습니다.
