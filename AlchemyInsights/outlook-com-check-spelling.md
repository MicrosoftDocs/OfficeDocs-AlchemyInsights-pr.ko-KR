# <a name="use-your-browser-to-check-spelling"></a>브라우저를 사용 하 여 맞춤법 검사

Outlook.com에 맞춤법 검사 기능이 포함 되어 있지는 않지만 대부분의 웹 브라우저에서 Microsoft Edge, Internet Explorer 10 이상 버전, 현재 버전의 Firefox, Chrome 및 Safari를 비롯 한 맞춤법 검사가 자동으로 수행 됩니다. 맞춤법 검사를 확인 하는 방법에 대 한 자세한 내용은 웹 브라우저의 옵션을 확인 하세요.
