# <a name="create-users-in-hybrid-deployments"></a>하이브리드 배포에서 사용자 만들기

조직에는 온-프레미스 환경과 Office 365 간의 하이브리드 배포 되 면 온-프레미스 Active Directory의 모든 사용자 개체를 만들 해야 합니다. Office 365 조직에 동기화는 자신이 하므로 적절 한 조직 구성 단위 (OU)에 사용자 개체를 만듭니다.
