# <a name="blocking-items-in-your-junk-email-folder"></a>정크 메일 폴더의 차단 항목

정크 메일 폴더에 이미 있는 항목을 차단 하면 해당 항목이 삭제 되지 않습니다. [정크 메일 폴더](https://outlook.live.com/mail/junkemail) 의 항목은 30 일 후에 자동으로 삭제 됩니다. 정크 메일 폴더에서 항목을 선택 하 여 삭제 되는 경우를 확인 합니다.

[보낸 사람을 차단 하거나 전자 메일을 정크 메일로 표시](https://support.office.com/article/a3ece97b-82f8-4a5e-9ac3-e92fa6427ae4)하는 방법에 대해 자세히 알아보세요.
