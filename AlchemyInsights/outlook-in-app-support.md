# <a name="in-app-support-in-outlook-for-windows"></a>Windows 용 Outlook의 앱 내 지원

테 넌 트 관리자는 Windows 용 Outlook의 문제에 대 한 앱 내 지원을 받을 수 있습니다. Outlook에서 **Help**  >  **지원 서비스에 문의**를 선택 합니다. 질문을 입력 하면 Outlook for Windows 지원 팀의 구성원이 메시지에 회신할 수 있습니다.
