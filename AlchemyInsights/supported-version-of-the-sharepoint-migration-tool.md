# <a name="supported-version-of-the-sharepoint-migration-tool"></a>지원 되는 버전의 SharePoint 마이그레이션 도구



현재 SharePoint 마이그레이션 도구는 SharePoint 2013의 마이그레이션만 지원 합니다. 지금은 버전 2010 및 2016가 지원 되지 않습니다.
  
자세한 내용은 [SharePoint 마이그레이션 도구 소개](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409)를 참조 하세요.
