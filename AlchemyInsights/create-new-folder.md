# <a name="create-a-new-folder"></a>새 폴더 만들기

1. 탐색 창의 **폴더** 에서 **새 폴더**를 선택 합니다. 
    
2. 폴더 이름을 입력 하 고 enter 키를 누릅니다.
    
자세한 내용은 [Outlook.com에서 폴더 만들기](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a)를 참조 하십시오.
