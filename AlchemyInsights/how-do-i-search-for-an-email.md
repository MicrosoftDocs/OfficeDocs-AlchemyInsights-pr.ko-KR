# <a name="how-do-i-search-for-an-email"></a>전자 메일을 검색 하는 방법

전자 메일을 보낸 연락처를 조회 하거나 전자 메일에 포함 된 키워드를 검색 하 여 전자 메일을 검색할 수 있습니다.

아래쪽 탐색 모음에서 돋보기를 누릅니다. 그런 다음 화면 맨 위의 상자에 검색 요청을 입력 합니다. 

**팁:** 돋보기를 두 번 탭 하 여 검색 상자에 포커스를 이동할 수 있습니다. 요청을 입력 하면 결과가 표시 됩니다.
