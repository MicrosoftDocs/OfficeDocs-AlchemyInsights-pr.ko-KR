# <a name="how-to-type-special-characters-on-a-mac"></a>Mac에 특수 문자를 입력 하는 방법

Mac에 특수 문자를 입력 하려면 키보드 바로 가기 ⌘ **명령** + **컨트롤** + **공간**을 사용 하 여 **특수 문자** 메뉴를 엽니다.

@의 빠른 바로 가기도 **ALT** + **64**입니다.
