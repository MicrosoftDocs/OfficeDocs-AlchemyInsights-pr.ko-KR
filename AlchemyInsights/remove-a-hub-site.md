# <a name="remove-a-hub-site"></a>허브 사이트 제거

새 SharePoint 관리 센터에서 허브 사이트 등록 취소
  
1. 새 SharePoint 관리 센터의 **사이트**에서 **활성 사이트**를 클릭 합니다. 
    
2. 사이트를 선택 하 고 **허브 사이트**를 클릭 한 다음 **허브 사이트로 등록 취소**를 클릭 합니다. 
    
3. **확인**을 클릭합니다. 
    
[허브 사이트에 대해 자세히 알아보기](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
