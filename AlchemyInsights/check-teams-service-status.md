# <a name="check-teams-service-status"></a>Teams 서비스 상태 확인

Microsoft Teams 서비스의 현재 상태를 확인하려면 Microsoft 365 관리 센터 서비스 상태 대시보드로 이동합니다.

자세한 내용은 [Microsoft 365 서비스 상태를 확인하는 방법](https://docs.microsoft.com/office365/enterprise/view-service-health)을 참조하세요.
