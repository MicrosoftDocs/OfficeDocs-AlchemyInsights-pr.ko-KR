# <a name="your-users-mailbox-is-being-created"></a>사용자의 사서함을 만들고 있습니다.

새 사서함을 사용할 준비가 되 면 최대 1 시간까지 걸릴 수 있습니다.
  
사용자의 사서함이 1 시간 넘게 "준비 중" 상태인 경우 사용자를 삭제 하 고 다시 만들거나 "내 전화를 드리겠습니다" 옵션을 사용 하 여 지원 엔지니어와 이야기할 수 있습니다.
