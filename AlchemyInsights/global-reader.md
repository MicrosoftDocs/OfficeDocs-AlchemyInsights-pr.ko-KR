# <a name="global-reader"></a>전역 읽기 권한자

이 역할에서 사용자는 Microsoft 365 서비스 전반에서 설정 및 관리 정보를 읽을 수 있지만 관리 작업을 수행할 수 없습니다. 전역 읽기 권한자는 전역 관리자의 읽기 전용 상대입니다.
OneDrive 관리 센터는 전역 읽기 권한자 역할을 지원하지 않습니다.

전역 읽기 권한자와 현재 제한에 대해 자세히 알아보려면 [전역 읽기 권한자](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#global-reader)를 참조하세요.
