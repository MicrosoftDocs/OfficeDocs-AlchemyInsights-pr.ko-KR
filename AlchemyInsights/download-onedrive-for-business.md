# <a name="download-onedrive-for-business"></a>비즈니스용 OneDrive 다운로드

Windows 10이 있는 경우 OneDrive가 PC에 이미 설치됩니다. 다른 버전의 Windows를 사용하는 경우 OneDrive를 설치하여 시작할 수 있습니다.

- 비즈니스용 OneDrive는 여기에서 다운로드할 수 [있습니다.](https://www.microsoft.com/microsoft-365/onedrive/download)
- 다운로드하여 설치하면 사용자에게 자신의 직장 또는 학교 계정에 로그인할지 묻는 요청이 표시됩니다.
- [OneDrive를 끄거나 사용하지 않도록 설정하거나 제거합니다.](https://support.microsoft.com/office/turn-off-disable-or-uninstall-onedrive-f32a17ce-3336-40fe-9c38-6efb09f944b0)
- Windows 또는 Mac에서 OneDrive와 [파일을](https://support.microsoft.com/office/615391c4-2bd3-4aae-a42a-858262e42a49) [동기화합니다.](https://support.microsoft.com/office/d11b9f29-00bb-4172-be39-997da46f913f)
