# <a name="trouble-completing-a-purchase-in-the-office-365-portal"></a>Office 365 포털의 구매를 완료 하는 문제

했을 때 된 하 라는 메시지가 여러 번 Office 365 포털에 로그인 하거나 문제가 발견 된 메시지를 수신 하는 경우를 구매를 확인 하려고 시도할 때, 브라우저의 inprivate/incognito 모드를 사용 하 여 시도 하는 최상의 첫번째 단계가입니다. Inprivate/incognito 창을 연 후 [portal.office.com](https://portal.office.com) 사이트로 이동 하 고 관리자 자격 증명을 사용 하 여 로그온 합니다. 구매를 완료 하려면 이동 합니다.
