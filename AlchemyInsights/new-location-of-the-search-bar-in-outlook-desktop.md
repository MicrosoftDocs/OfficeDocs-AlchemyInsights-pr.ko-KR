# <a name="new-location-of-the-search-bar-in-outlook-desktop"></a>Outlook 데스크톱에서 검색 창의 새 위치

다른 Office 응용 프로그램과 일치하도록 Outlook 데스크톱 검색 상자가 제목 표시줄로 이동하였습니다. 이 새 검색 홈은 추가 메시지가 들어갈 공간이 포함된 더 깔끔한 메시지 목록을 남길 것입니다.
