# <a name="copy-files-in-sharepoint-and-onedrive"></a>SharePoint 및 OneDrive에서 파일 복사

웹 브라우저에서 SharePoint 및 OneDrive 내의 다른 위치로 파일을 복사할 수 있습니다.

자세한 내용은 이동 또는 [복사를 참조하세요.](https://support.microsoft.com/office/00e2f483-4df3-46be-a861-1f5f0c1a87bc)

> [!NOTE]
> SharePoint **Online의** 클래식 환경을 사용하는 경우 복사를 사용할 수 없습니다.
