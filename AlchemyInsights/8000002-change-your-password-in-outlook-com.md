# <a name="change-your-password"></a>사용자 암호 변경

Outlook.com 암호 Microsoft 계정 암호와 동일 합니다.
  
암호를 변경 하려면로 이동 [https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409) 하 고 **암호 변경**을 선택 합니다. 
  
암호를 잊으셨습니까 하는 경우 [암호 페이지를 다시 설정 하는 Microsoft 계정](https://go.microsoft.com/fwlink/p/?linkid=841909)으로 이동 합니다.
  
자세한 내용은, 참조 [Outlook.com에 암호를 변경](https://go.microsoft.com/fwlink/?linkid=873109)합니다.
