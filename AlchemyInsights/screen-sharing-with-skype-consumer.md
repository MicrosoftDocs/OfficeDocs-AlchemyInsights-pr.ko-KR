# <a name="screen-sharing-with-skype-consumer-users"></a>Skype 소비자 사용자와 화면 공유

프로그램을 공유하는 방법, 공유 제한에 대해 알아보고 리소스 문제 해결에 대한 일반적인 지침은 비즈니스용 Skype에서 화면 [공유를 검토하세요.](https://support.microsoft.com/office/share-and-present-content-from-skype-meetings-app-skype-for-business-web-app-234b0c06-a88d-4707-904c-4fd6c571fc01)  

현재 비즈니스용 Skype 사용자는 Skype 소비자와의 공유를 화면화할 수 없습니다. 소비자 [Skype 사용자와](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) 화면 공유를 사용하도록 설정하는 데 관한 피드백을 제공하려면 여기를 클릭하세요.
