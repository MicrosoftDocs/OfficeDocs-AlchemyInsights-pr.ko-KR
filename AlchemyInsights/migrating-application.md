# <a name="migrating-applications"></a>응용 프로그램 마이그레이션

[응용 프로그램을 Azure Active Directory로](https://docs.microsoft.com/azure/active-directory/manage-apps/migration-resources) 마이그레이션하기 위한 리소스 - Azure AD(Azure Active Directory)로 응용 프로그램 액세스 및 인증을 마이그레이션하는 데 도움이 되는 리소스입니다.

[Active Directory Federation Services에서 Azure Active Directory로](https://docs.microsoft.com/azure/active-directory/manage-apps/migrate-adfs-apps-to-azure) 응용 프로그램 인증 이동 - 개발자는 Active Directory Federation Services에서 Azure Active Directory로 응용 프로그램을 마이그레이션하기 위한 지침을 중점적으로 설명합니다.
