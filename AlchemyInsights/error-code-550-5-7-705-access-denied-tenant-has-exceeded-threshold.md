# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 액세스 거부, 테 넌 트 임계값을 초과한

이 메시지는 Office 365 조직에서 너무 많은 스팸 또는 대량 메일을 보낸 경우에 발생 합니다. 이 블록을 제거 하려면 이해 하 고는 지원 에이전트에 대 한 원인을 설명으로 원본으로 사용 되는 문제를 해결 해야 합니다.
