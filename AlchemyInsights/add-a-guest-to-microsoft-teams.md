# <a name="add-a-guest-to-microsoft-teams"></a>Microsoft 팀에 게스트 추가

팀 조직에 없는 사용자와 공동 작업을 하려는 경우가 있습니다. 전자 메일에서이 작업을 수행 하는 경우가 많지만 팀에서 공동 작업을 하려는 경우에는이를 게스트로 추가할 수 있습니다.

자세한 내용은 팀 [의 팀에 게스트 추가](https://support.office.com/article/add-guests-to-a-team-in-teams-fccb4fa6-f864-4508-bdde-256e7384a14f#ID0EAABAAA=Desktop)를 참조 하세요.

**참고:** 팀 소유자만이 게스트를 추가할 수 있습니다 ( [게스트 권한 설정](https://support.office.com/article/set-guest-permissions-for-channels-in-teams-4756c468-2746-4bfd-a582-736d55fcc169)참조).
