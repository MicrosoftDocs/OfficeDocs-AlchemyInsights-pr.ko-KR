# <a name="delete-a-sharepoint-page"></a>SharePoint 페이지 삭제

1. 해당 페이지가 있는 사이트로 이동 합니다.
    
2. 왼쪽에서 **사이트 콘텐츠**를 클릭 합니다.
    
3. 사이트 콘텐츠 페이지의 왼쪽에서 **페이지**를 클릭 합니다.
    
4. 페이지를 선택 하 고 페이지 오른쪽에 있는 줄임표 (...)를 클릭 합니다.
    
5. **삭제**를 클릭합니다.
