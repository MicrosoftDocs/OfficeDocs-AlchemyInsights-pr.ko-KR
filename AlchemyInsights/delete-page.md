# <a name="delete-a-sharepoint-page"></a>SharePoint 페이지 삭제

1. 여기서는 페이지는 사이트로 이동 합니다.
    
2. 왼쪽된 가장자리에서 **사이트 콘텐츠**를 클릭 합니다.
    
3. 사이트 콘텐츠 페이지의 왼쪽된 가장자리에서 **페이지**를 클릭 합니다.
    
4. 페이지를 선택한 다음 페이지의 오른쪽에 줄임표 (...)를 클릭 합니다.
    
5. **삭제**를 클릭합니다.
