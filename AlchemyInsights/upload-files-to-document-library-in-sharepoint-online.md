# <a name="upload-files-to-onedrive-or-sharepoint"></a>OneDrive 또는 SharePoint에 파일 업로드

컴퓨터에서 파일을 끌어 문서 라이브러리에 놓는 방법으로 SharePoint Online에서 문서 라이브러리에 업로드할 수 있습니다. OneDrive를 사용 하 여 컴퓨터와 클라우드 간에 파일을 동기화 하 여 어디에서 든 지 파일에 액세스할 수 있습니다. 또한 파일 탐색기에서 직접 동기화 된 파일을 사용 하 여 작업을 수행 하 고 오프 라인일 때도 파일에 액세스할 수 있습니다. 업로드 및 동기화에 대 한 자세한 내용을 보려면 아래 링크를 방문 하세요.

- [라이브러리에 폴더 또는 파일 업로드](https://support.office.com/article/upload-a-folder-or-files-to-a-document-library-eb18fcba-c953-4d45-8d90-8da66edeacdb)

- [동기화를 사용 하 여 업로드](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-in-windows-615391c4-2bd3-4aae-a42a-858262e42a49)

- [파일 제한 (잘못 된 문자, 파일 업로드 크기, 경로 길이 등)](https://support.office.com/article/invalid-file-names-and-file-types-in-onedrive-onedrive-for-business-and-sharepoint-64883a5d-228e-48f5-b3d2-eb39e07630fa)

- [OneDrive 동기화 문제 해결](https://support.office.com/article/Fix-OneDrive-sync-problems-83ab0d8a-8400-45b0-8dcf-dc8aa8a6bcf8)

- [OneDrive 웹 사이트에서 파일을 업로드 하는 문제 해결](https://support.office.com/article/Fix-problems-uploading-files-on-the-OneDrive-website-9afcc4a0-e344-4bc9-9c9d-59d3e802247e)

- [업로드 및 다운로드 속도 최대화](https://support.office.com/article/Maximize-upload-and-download-speed-8eeadfb8-501f-406d-997b-98ab6ff67f43)
