# <a name="add-users-to-a-sharepoint-group"></a>SharePoint 그룹에 사용자 추가

SharePoint 그룹은 사이트 및 콘텐츠에 대 한 사용 권한 집합이 모두 동일한 사용자 모음입니다. 한 번에 한 사람에 게 사용 권한을 할당 하는 대신 그룹을 사용 하 여 여러 사용자에 게 한 번에 동일한 사용 권한 수준을 간편 하 게 할당할 수 있습니다.

- [그룹에 사용자 추가](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#add-users-to-a-group)
- [관리 센터를 사용 하 여 Office 365 그룹에서 구성원 추가 또는 제거](https://docs.microsoft.com/office365/admin/create-groups/add-or-remove-members-from-groups?view=o365-worldwide)
