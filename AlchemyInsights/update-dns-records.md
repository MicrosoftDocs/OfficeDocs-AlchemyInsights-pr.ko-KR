# <a name="update-your-dns-records"></a>DNS 레코드를 업데이트 합니다.

Office 365 사용 (영문) 사용자 지정 도메인을 가져오려면 DNS 레코드를 업데이트 하려면 필요 합니다. 자세한 지침을 보려면 다음 링크를 사용 합니다.
  
 **도움말 문서** (등록자에 대 한 모든 도메인) 
  
- [DNS 호스팅 공급자에 상관없이 Office 365용 DNS 레코드 만들기](https://docs.microsoft.com/office365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)
    
 **비디오 가이드** (도메인에 대 한 GoDaddy.com에서 호스트) 
  
- [도메인 및 사용자 설정](https://support.office.com/article/26524a2c-1d65-48ab-8927-ae0b27370c62)
