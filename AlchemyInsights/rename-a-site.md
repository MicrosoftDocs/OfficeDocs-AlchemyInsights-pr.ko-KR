# <a name="rename-a-site"></a>사이트 이름 바꾸기

예를 들어 사이트의 이름을 https://contoso.sharepoint.com/sites/sample2 에서  [](https://go.microsoft.com/fwlink/?Linkid=2018691) 으로 변경하려면, 새 사이트를 만든 다음 기존 사이트에서 새 사이트로 모든 콘텐츠를 이동해야 합니다. 문서 라이브러리에서 항목을 이동하는 방법에 대한 정보는 SharePoint 문서 라이브러리에서 항목을 이동 또는 복사를 참조하세요.
  
참고: SharePoint Online URL에서 도메인 이름 (즉, 위의 URL의 contoso 부분)을 변경 하려는 경우는 불가능 합니다. [사용자 도메인 변경](https://go.microsoft.com/fwlink/?Linkid=2018696)을 참조 하십시오.
