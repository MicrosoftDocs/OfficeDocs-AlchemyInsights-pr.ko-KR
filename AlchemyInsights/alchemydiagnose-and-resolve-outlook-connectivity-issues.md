# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>문제를 진단 하 고 Outlook 연결 문제를 해결 합니다.

문제를 진단 하 고 Outlook 연결 문제를 해결 하기 위해 영향을 받는 컴퓨터에서 [Outlook 연결 도구](https://aka.ms/SaRA-OutlookDisconnect) 를 실행할 수 있습니다. 
  
이 도구는 진단 검사를 수행 하 고 Outlook 연결 문제를 해결 하는 경우에 대 한 가능한 해결 방법을 제안 합니다.
