# <a name="change-billing-country-or-region"></a>청구 국가 또는 지역 변경

Microsoft 365 관리 센터의 청구 청구 계정에서 국가/지역을   >   변경할 수 없습니다. 새 계정을 등록하고 원하는 국가/지역을 선택해야 합니다.
