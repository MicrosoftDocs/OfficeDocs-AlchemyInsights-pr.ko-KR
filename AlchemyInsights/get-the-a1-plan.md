# <a name="get-the-a1-plan"></a>A1 플랜 받기

Microsoft 365 교육용 교육 기관에 이미 자격이 있는 경우 **Microsoft 365 관리 센터**에서 A1 구독을 추가할 수 있습니다. 

1. **청구 > [제품 및 서비스](https://go.microsoft.com/fwlink/p/?linkid=868433)** 로 이동합니다.

2. 검색 상자를 사용하여 "A1"을 찾으세요.

3. A1 플랜을 사용할 사용자 수를 선택하세요.

기관에서 아직 Microsoft 365 Education을 설정하지 않았으나 Office 365를 무료로 추가하려는 경우 전자 메일을 제공하여 [시작](https://www.microsoft.com/education/products/office)할 수 있습니다.
