# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>하이브리드 환경에서 일정 약속 있음/없음 문제

사용자가 일정 약속 있음/없음 정보 하이브리드 환경에서 특정인에 문제가 있을 경우 하이브리드 구성 마법사를 다시 실행 합니다. 하이브리드 구성 마법사에 대 한 자세한 내용은 [하이브리드 구성 마법사](https://go.microsoft.com/fwlink/p/?linkid=528149)를 참조 하십시오.
  
단계 하이브리드 환경에서의 문제를 해결 하는 것에 대 한 [해결 하이브리드 배포를](https://technet.microsoft.com/library/jj659053.aspx)참조 하십시오.
