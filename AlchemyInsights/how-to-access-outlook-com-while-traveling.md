# <a name="how-to-access-outlookcom-while-traveling"></a>여행 중에 Outlook.com에 액세스 하는 방법

여행 중에 Outlook.com에 액세스 하려면 Microsoft 계정 보안 정보-보안 코드를 받을 수 있는 대체 전자 메일 주소 또는 전화 번호가 최신 상태 인지 확인 합니다. 자세한 내용은 [여행 시 Outlook.com에 액세스 하는 방법을](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy)참조 하세요.
