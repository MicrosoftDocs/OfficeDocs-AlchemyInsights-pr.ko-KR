# <a name="getting-a-tenantaccessblockedexception-error-when-accessing-email"></a>전자 메일에 액세스할 때 TenantAccessBlockedException 오류를 사용 합니까?

테 넌 트에 대 한 액세스가 차단가 없다는 오류가 표시 얻을 수 있는 이유는 몇 가지 이유가 있습니다. 이 문제를 해결 하려면 다음 단계를 시킬 수 있습니다.
  
- 구독에 [구독](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/subscriptions) 페이지에서 유효한 지 확인 합니다. 
    
- [대금 청구](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/billoverview) 페이지를 확인 했어야 과거의 모든 사용자 계정에 있습니다. 
    
- 계정은 순서로 되지만 최근에 변경 된 경우 변경 내용 모든 서버로 전파 하려면 시간이 다소 걸릴 수 있습니다. 잠시 후에 다른 시도 지정 하 고 문제가 지속 되 면 문의처 키를 누릅니다.
