1. <span data-ttu-id="f61d0-103">Office 365 보안 및 준수 센터 > **분류** > **민감도 레이블** 로 이동합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-103">Go to the Office 365 Security & Compliance Center > **Classification** > **Sensitivity labels**.</span></span>

1. <span data-ttu-id="f61d0-104">민감도 레이블 페이지에서 **레이블 정책** 탭을 선택하고 **레이블 게시** 를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-104">On the Sensitivity labels page, select the **Label policies** tab, and then select **Publish labels**.</span></span>

1. <span data-ttu-id="f61d0-105">**민감도 레이블을 선택하여 게시** 를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-105">Select **Choose sensitivity labels to publish**.</span></span> <span data-ttu-id="f61d0-106">앱과 서비스에서 사용할 수 있도록 설정할 레이블을 선택한 다음 **추가** 를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-106">Select the labels that you want to make available in apps and to services, and then select **Add**.</span></span>

    <span data-ttu-id="f61d0-107">**중요**: 하위 레이블을 선택한 경우 해당 상위 레이블도 선택해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-107">**Important**: If you select a sublabel, make sure you also select its parent label.</span></span>

1. <span data-ttu-id="f61d0-108">선택한 레이블을 검토합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-108">Review the selected labels.</span></span> <span data-ttu-id="f61d0-109">변경하려면 **편집** 을 선택하고 변경하지 않으려는 경우, **다음** 을 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-109">To make any changes, select **Edit**; otherwise, select **Next**.</span></span>

1. <span data-ttu-id="f61d0-110">프롬프트에 따라 정책 설정을 구성합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-110">Follow the prompts to configure the policy settings.</span></span>

1. <span data-ttu-id="f61d0-111">여러 사용자나 위치에 대해 서로 다른 정책 설정이 필요한 경우에는 이 단계를 반복합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-111">Repeat these steps if you need different policy settings for different users or locations.</span></span> <span data-ttu-id="f61d0-112">예를 들어 사용자 그룹에 대한 추가 레이블이나 사용자 하위 집합에 대해 다른 기본 레이블을 원하는 경우가 해당됩니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-112">For example, you want additional labels for a group of users, or a different default label for a subset of users.</span></span>

1. <span data-ttu-id="f61d0-113">사용자 또는 위치에 충돌이 발생할 수 있는 레이블 정책을 여러 개 만드는 경우에는 정책 순서를 검토하고 필요한 경우 해당 항목을 위나 아래로 이동합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-113">If you create more than one label policy that might result in a conflict for a user or location, review the policy order and if necessary, move them up or down.</span></span> <span data-ttu-id="f61d0-114">레이블 정책 순서를 변경하려면 **추가 작업** 에 대해 **...** 를 선택한 다음 **위로 이동** 또는 **아래로 이동** 을 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-114">To change the order of a label policy, select **...** for **More actions**, and then select **Move up** or **Move down**.</span></span>

<span data-ttu-id="f61d0-115">마법사를 완료하면 레이블 정책이 자동으로 게시됩니다.</span><span class="sxs-lookup"><span data-stu-id="f61d0-115">Completing the wizard automatically publishes the label policy.</span></span>
