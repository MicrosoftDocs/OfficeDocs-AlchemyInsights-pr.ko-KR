# <a name="endpoint-dlp-licensing-error"></a>끝점 DLP 라이선스 오류

끝점 DLP를 설정 하는 동안 다음 오류가 표시 되는 경우:

`Your organization is missing the licenses required to manage these devices`.

다음 구독 또는 추가 기능 중 하나를 사용 하 고 있는지 확인 합니다.

- Microsoft 365 E5
- Microsoft 365 A5(EDU)
- Microsoft 365 E5 compliance
- Microsoft 365 A5 compliance
- Microsoft 365 E5 Information Protection 및 거버넌스
- Microsoft 365 A5 Information Protection 및 거버넌스

> [!NOTE]
> 이는 Win E5 + O365 E5 + EMS E5와 같은 라이선스 조합에서는 작동 하지 않습니다. 이 기능을 설치 하려면 순수한 M365 E5 라이선스가 있어야 합니다.

끝점 DLP 라이선스 정보에 대 한 자세한 내용은 [ENDPOINT DLP license](https://docs.microsoft.com/microsoft-365/compliance/endpoint-dlp-getting-started#onboarding-devices-into-device-management) 를 참조 하십시오.
