# <a name="sharing-options-are-grayed-out"></a>공유 옵션이 회색으로 표시 됨

SharePoint Online 또는 OneDrive에서 공유 하려고 할 때 특정 공유 옵션이 회색으로 표시 되는 경우 [SharePoint Online 또는 OneDrive에서 공유할 때 공유 옵션이 회색으로 표시 되는 경우](https://docs.microsoft.com/sharepoint/support/administration/sharing-options-grayed-out-when-sharing-from-sharepoint-online-or-onedrive)를 검토 하세요.
