# <a name="how-to-right-click-on-a-tablet"></a>태블릿에서 마우스 오른쪽 단추를 클릭 하는 방법

서피스나 Windows 태블릿을 마우스 오른쪽 단추로 클릭 하 고 몇 초 동안 선택 영역을 길게 누릅니다. 릴리스를 하면 마우스 오른쪽 단추를 클릭 하는 메뉴가 나타납니다.
