# <a name="how-to-change-your-theme"></a>테마를 변경 하는 방법

Outlook.com 또는 웹에서 Outlook에서 테마를 변경 하려면 **설정**으로 이동![설정](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) **테마**에서 테마를 선택 합니다. 사용 가능한 모든 테마를 보려면 **테마**아래에 있는 **모든 보기** 를 선택 합니다.
