# <a name="delete-a-contact"></a>연락처 삭제

1. 페이지의 왼쪽 아래 모서리에서 **사용자** 아이콘을 선택 합니다.
2. 삭제할 연락처를 선택 하 고 **삭제**를 선택 합니다.

**삭제**를 선택할 수 없는 경우 연락처가 Skype 또는 연결 된 소셜 네트워크 계정에 있는 것일 수 있습니다. 연락처를 삭제 하려면 Skype 또는 연락처가 있는 계정으로 이동 합니다.
