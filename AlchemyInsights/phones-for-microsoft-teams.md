# <a name="are-you-looking-for-phones-that-are-compatible-with-teams"></a>팀과 호환 되는 전화를 찾으십니까?

관리자는 Microsoft 365 관리 센터에서 조직의 팀과 함께 사용 되는 [모든 장치를 관리](https://docs.microsoft.com/microsoftteams/device-management) 합니다. 자세한 내용은 다음을 참조하세요. 

- [Microsoft 팀의 전화](https://docs.microsoft.com/microsoftteams/phones-for-teams)목록을 검토 합니다. 
- [현재 인증 된 IP 전화](https://docs.microsoft.com/microsoftteams/teams-ip-phones#currently-certified-ip-phones)목록을 검토 합니다. 
- [알려진 제한 사항 및 문제가 있는 장치](https://support.office.com/article/control-calls-using-a-headset-in-teams-65d6e104-444d-4013-b8c2-f11317dd69a8)목록을 검토 합니다.
