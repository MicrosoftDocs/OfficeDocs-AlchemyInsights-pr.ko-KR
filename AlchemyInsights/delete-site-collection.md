# <a name="delete-a-sharepoint-site-collection"></a>SharePoint 사이트 모음 삭제

1. 사이트 모음 페이지 클래식 SharePoint 관리 센터에서 사이트 모음 옆에 있는 확인란을 선택 합니다.
    
2. 사이트 모음 탭에서 **삭제**를 선택 합니다.
