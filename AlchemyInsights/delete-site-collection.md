# <a name="delete-a-sharepoint-site"></a>SharePoint 사이트 삭제

1. 새 SharePoint 관리 센터의 사이트 페이지에서 삭제할 사이트 옆에 있는 확인란을 클릭 합니다.
    
2. 리본 도구 모음에서 **삭제**를 클릭 합니다.
