# <a name="stream-quality"></a>Stream 품질

Stream은 HTTPS 유니캐스트 ABR(적응 비트 전송률 스트리밍)을 사용하여 사용 가능한 네트워크 대역폭 및 비디오 플레이어의 크기를 기준으로 최상의 비디오 재생 품질을 동적으로 선택합니다. 자세한 내용은 다음을 참조하세요.

- [Microsoft Stream 비디오 제공 및 네트워크 개요](https://docs.microsoft.com/stream/network-overview)
