# <a name="having-issues-getting-outlook-to-connect"></a>Outlook 연결 시작 관련 문제가 있습니까?

Outlook 다음 도메인에서 제대로 작동 하도록 구성 되지 않았습니다는 하는데이 했을 때:
  
{HealthCheckString}
  
NS 또는 CNAME 레코드 (DNS) 도메인 설정을 올바르게 설정 되지 않습니다. 자세한 내용은 [Office 365의 업데이트 DNS 레코드](https://support.office.com/article/https://support.office.com/en-us/article/Create-DNS-records-for-Office-365-when-you-manage-your-DNS-records-B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23.aspx) 를 참조할 수 있습니다. 
  
를 해결 하려면 Office 365 관리 센터에서 [도메인](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/Domains) 페이지로 이동 하십시오 하 고 함께 알아봅니다. 
  
다른 일반적인 솔루션에 대 한 [도메인 FAQ를 확인](https://support.office.com/article/https://support.office.com/article/7b7b075d-79f9-4e37-8a9e-fb60c1d95166.aspx) 합니다.
