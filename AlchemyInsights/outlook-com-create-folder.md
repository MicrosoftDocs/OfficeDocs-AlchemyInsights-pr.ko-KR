# <a name="create-a-folder-in-outlookcom"></a>Outlook.com에서 폴더 만들기

1. 폴더 창의 맨 아래에서 **새 폴더**를 선택 합니다. 폴더 창에서 아래로 스크롤하여 **새 폴더** 단추를 확인 해야 할 수 있습니다.
2. 텍스트 상자에 폴더 이름을 입력 한 다음 enter 키를 누릅니다.

자세한 내용은 [Outlook.com에서 폴더 만들기](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)를 참조 하십시오.
