# <a name="admin-roles"></a>관리자 역할

- 관리 센터에서 [역할](https://admin.microsoft.com/Adminportal/Home#/roles)을 선택하여 관리자 역할을 지정하거나 자세한 내용을 알아보세요.

- 그런 다음 역할 이름을 선택하고 **역할 할당**을 클릭하거나 **역할 비교**를 클릭하여 자세히 알아보세요.
