# <a name="cant-download-files-in-chrome"></a>Chrome에서 파일을 다운로드할 수 없음

**문제** Google Chrome 브라우저를 사용할 때 일부 고객이 파일을 다운로드할 수 없도록 하는 문제에 대해 알아봅니다. 

**해결 방법** [Microsoft Edge](https://www.microsoft.com/en-gb/windows/microsoft-edge) 또는 다른 웹 브라우저를 사용 합니다.
파일을 다운로드 하는 방법에 대 한 자세한 내용은 [Outlook.com attachments](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy)를 참고 하십시오.
