# <a name="change-your-profile-name-in-outlookcom"></a>Outlook.com에서 사용자 프로필의 이름을 변경 합니다.

1. [Your Microsoft 계정 정보](https://go.microsoft.com/fwlink/p/?linkid=860841)로 이동 합니다.
    
2. **이름을 편집**을 선택 합니다. 
    
3. 이름 및 성, 이름, 입력 하 고 **저장**을 선택 합니다. 
    
프로필 사진 또는 효과로 이동 하는 이름에 대 한 변경 24 시간까지 기다리십시오.
