# <a name="subscription-needed-for-microsoft-edge-sync-to-work-with-azure-active-directory-accounts"></a>Microsoft Edge 동기화가 Azure Active Directory 계정과 작동하기 위해 필요한 구독

동기화는 다음 구독 중 하나에서 작동됩니다.

1. Azure Active Directory Premium(P1 및 P2)
1. Microsoft 365 Business Premium
1. Office 365 E3 이상
1. Azure Information Protection(P1 및 P2)
1. 모든 Office 365 Education 구독:
    - 학생 또는 교직원용 Microsoft 앱
    - 학생 또는 교직원용 Exchange Online
    - Office 365 A1 이상
    - Microsoft 365 A1 이상
    - 학생 또는 교직원용 Azure Information Protection P1 또는 P2
