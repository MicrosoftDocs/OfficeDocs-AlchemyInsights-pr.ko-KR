# <a name="create-a-sensitivity-label"></a>민감도 레이블 만들기

민감도 레이블을 만들 경우 다음 단계를 수행합니다.

1. **[Office 365 보안 및 준수 센터](https://sip.protection.office.com/) > 분류 > 민감도 레이블** 로 이동합니다.
2. **+레이블 만들기** 를 선택하고 **새 민감도 레이블** 마법사를 시작합니다. 람호화 설정 및 자동 레이블 지정에 대한 자세한 내용은 [민감도 레이블에서 암호화를 사용하여 콘텐츠 액세스 제한](https://docs.microsoft.com/microsoft-365/compliance/encryption-sensitivity-labels) 및 [자동으로 콘텐츠에 민감도 레이블 적용](https://docs.microsoft.com/microsoft-365/compliance/apply-sensitivity-label-automatically)을 참조하세요.
3. 이 단계를 반복하여 레이블을 더 만듭니다. 그러나 하위 레이블을 만들려는 경우 먼저 상위 레이블을 선택하고 **추가 작업** 에서 **...** 을 클릭한 다음 **하위 레이블 추가** 를 클릭합니다.
4. 필요한 레이블을 모두 만든 경우 해당 주문을 검토하고 필요한 경우 해당 항목을 위나 아래로 이동합니다. 레이블 순서를 변경하려면 **추가 작업** 에 대해 **...** 를 클릭한 다음 **위로 이동** 또는 **아래로 이동** 을 클릭합니다.
