# <a name="i-dont-see-automatic-replies"></a>자동 응답이 표시되지 않습니다.

파일을 선택한 후 자동 응답이 표시되지 않는 경우 Outlook 자동 응답 기능을 지원하지 않는 Gmail, Yahoo 또는 기타 POP 또는 IMAP 계정을 사용하고 있는 것일 수 있습니다. 들어오는 메시지에 회신하는 규칙을 설정할 수 있지만 Outlook을 실행 중인 경우만 설정할 수 있습니다.
