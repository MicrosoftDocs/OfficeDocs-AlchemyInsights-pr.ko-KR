# <a name="shared-mailbox-or-calendar-issue-in-outlook-for-mac"></a>Mac 용 Outlook의 공유 사서함 또는 일정 문제 인가요?

응용 프로그램에서 직접 연락 하는 경우 더 빠르게 작업할 수 있습니다. **영향을 받는 사용자의 Mac로 이동한 후 Outlook을 열고 도움말 \> 지원 서비스를 클릭**합니다. 
  
Mac 용 outlook for app 지원을 사용 하는 방법에 대 한 자세한 내용은이 문서: [2016 outlook의 새 연락처 지원 기능 For mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx) 를 참조 하세요.
