# <a name="what-to-do-if-you-cant-create-a-team-in-microsoft-teams"></a>Microsoft Teams에서 팀을 만들 수 없는 경우 할 일

Teams 관리자에게 다음을 요청합니다.

1. Microsoft 365에서 조직의 보안 그룹 설정을 만들거나 변경합니다.
2. 해당 보안 그룹에 추가합니다.
