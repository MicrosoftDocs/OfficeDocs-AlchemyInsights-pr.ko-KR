# <a name="disabling-advanced-outlookcom-security"></a>고급 Outlook.com 보안을 사용 하지 않도록 설정

사용자 계정에 대 한 최상의 보호 기능을 제공 하려면 Outlook.com 메시지에 대 한 모든 링크 피싱 메일 가져오거나 바이러스 또는 사용자의 컴퓨터에 맬웨어를 다운로드 하는 경우가 있는지 여부를 확인 합니다. 이 고급 Outlook.com 보안 제거 하 시겠습니까 인 경우 **예** 아래 클릭 하십시오.
