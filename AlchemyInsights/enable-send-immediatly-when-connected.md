# <a name="enable-send-immediately-when-connected"></a>연결 되 면 즉시 보내기 사용
 
1. 파일 탭에서 **옵션**을 클릭합니다.

2. Outlook 옵션 대화 상자에서 **고급**을 클릭 합니다.

3. 보내기 및 받기 섹션에서 **연결 되 면 즉시 보내기를**사용 하도록 설정 합니다. **확인**을 클릭합니다.

자세한 내용은 다음을 참조 하십시오.
