# <a name="public-folder-access-from-mobile-devices"></a>모바일 장치에서 든 지 공용 폴더 액세스

모바일 장치에서 콘텐츠에 액세스할 수 있고 공용 폴더와 유사한 기능을 포함 하는 것이 싶으면, 더 나은 대체에 대 한 [Office 365 그룹에 대 한 설명](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) 를 참조 하십시오.
