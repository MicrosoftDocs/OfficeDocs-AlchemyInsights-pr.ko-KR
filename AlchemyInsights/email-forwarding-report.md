# <a name="email-forwarding-report"></a>보고서를 전달 하는 전자 메일

조직에 얼마나 많은 자동 전달 메시지에서 보고서를 보려면, 체크인 [메일 흐름 인 사이트](https://support.office.com/article/beb6acaa-6016-4d54-ba7e-3d6d035e2b46) 는 [Office 365 보안 &amp; 준수 센터](https://protection.office.com/#/homepage)합니다.
  
참고: 처음 보안에 로그인 하는 경우 &amp; 준수 센터의 사용 가능한 옵션을 모두 표시 하려면 몇 분 후 브라우저를 새로 고칠 해야 할 수도 있습니다.
