# <a name="find-the-status-of-your-microsoft-365-ticket"></a>Microsoft 365 티켓 상태 확인

티켓에서 상태 업데이트를 찾고 있다면 관리 센터에서 티켓 상태를 볼 수 있습니다. 관리 센터에서 **모두 표시 > 지원**를 클릭한 다음 **서비스 요청 보기**을 클릭합니다.
