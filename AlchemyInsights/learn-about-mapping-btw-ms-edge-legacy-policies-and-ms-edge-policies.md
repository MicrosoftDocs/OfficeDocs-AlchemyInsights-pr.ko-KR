# <a name="learn-about-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Microsoft Edge 레거시 정책과 Microsoft Edge 정책 간의 매핑에 대해 자세히 알아보십시오.

Microsoft Edge 레거시(버전 45 및 이전 버전) 및 MDM(모바일 장치 관리) 설정의 그룹 정책은 Microsoft Edge 버전 80의 관련 정책에 매핑됩니다. 자세한 내용은 [Microsoft Edge 레거시에서 Microsoft Edge 정책 매핑을 참조하세요.](https://go.microsoft.com/fwlink/?linkid=2141665)
