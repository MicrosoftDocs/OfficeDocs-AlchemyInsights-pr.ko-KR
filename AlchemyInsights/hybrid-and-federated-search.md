# <a name="hybrid-and-federated-searches"></a>하이브리드 및 페더레이션된 검색 

SharePoint Online의 하이브리드 및 연결 된 검색에 대 한 자세한 내용은 [sharepoint의 하이브리드 검색](https://docs.microsoft.com/sharepoint/hybrid/hybrid-search-in-sharepoint)을 참조 하세요.

하이브리드 연결 된 검색을 계획 하는 방법에 대 한 자세한 내용은 [Plan hybrid 페더레이션된 search For SharePoint Server](https://docs.microsoft.com/sharepoint/hybrid/plan-hybrid-federated-search)을 참조 하세요.
