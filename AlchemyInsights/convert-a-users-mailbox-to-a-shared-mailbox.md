# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>사용자 사서함을 공유 사서함으로 변환

개별 사용자의 사서함을 공유 사서함으로 변환 하면 기존 전자 메일 및 일정 항목이 여러 사용자가 액세스할 수 있는 공유 사서함에 보존 됩니다. 공유 사서함을 사용자 (개인) 사서함으로 다시 변환할 수도 있습니다.
