# <a name="learn-about-external-calendar-sharing-options"></a>외부 일정 공유 옵션에 대 한 자세한 정보

조직 외부의 사용자와 모임을 예약 해야 하는 경우 Microsoft 365을 사용 하면 약속 있음/없음 시간을 확인 해야 하지만 Microsoft 365 환경에 대 한 사용자 계정이 없는 "외부 사용자"에 게 캘린더를 사용할 수 있습니다. 자세한 내용은 [외부 사용자와 일정 공유](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users)를 참조 하세요.
