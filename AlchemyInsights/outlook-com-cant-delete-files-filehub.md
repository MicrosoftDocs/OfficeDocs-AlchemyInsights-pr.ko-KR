# <a name="cant-delete-files-or-photos-from-files"></a>파일 또는 사진을 삭제할 수 없음

파일 사서함의 메시지에 첨부 된 파일 및 사진을 표시 합니다. 파일에서 직접 파일 또는 사진을 삭제할 수는 없습니다. 파일 및 사진을 삭제 하려면 첨부 된 전자 메일 메시지를 삭제 해야 합니다.

자세한 내용은 [Outlook.com에서 파일 또는 사진 삭제 방법을](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx)참조 하세요.
