# <a name="turn-external-sharing-on-or-off-for-a-site"></a>사이트에 대한 외부 공유 설정 또는 해제

사이트에 대 한 외부 공유 설정 (이전에는 "사이트 모음" 이라고 함)을 변경 하려면 전역 또는 SharePoint 관리자 여야 합니다. 사이트 소유자는 이 설정을 변경할 수 없습니다. 

자세한 내용은 [사이트에 대 한 외부 공유 설정 또는 해제](https://docs.microsoft.com/sharepoint/change-external-sharing-site)를 참조 하세요.
