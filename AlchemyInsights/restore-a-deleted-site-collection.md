# <a name="restore-a-deleted-site"></a>삭제된 사이트 복원

관리자가 사이트를 삭제 하면 사이트가 영구적으로 삭제 되기 전까지 93 일 전에 보관 되는 사이트 모음 휴지통에 저장 됩니다. 사이트를 복원 하려면
  
1. 새 SharePoint 관리 센터에서 리본 메뉴의 **휴지통** 을 클릭 합니다. 
    
2. 복원 하려는 사이트 모음 옆에 있는 확인란을 선택 합니다.
    
3. **지운 편지함 복원을**클릭 합니다.
    
삭제 된 통신 사이트를 복원 하려면 새 SharePoint 관리 센터를 사용할 수 있습니다. 그렇지 않으면 Microsoft PowerShell을 사용 해야 합니다. Office 365 그룹에 속하는 사이트를 복원 하려면 Exchange 관리 센터에서 그룹을 복원 해야 합니다. 그룹을 삭제 한 후 30 일 동안 복원할 수 있습니다.
