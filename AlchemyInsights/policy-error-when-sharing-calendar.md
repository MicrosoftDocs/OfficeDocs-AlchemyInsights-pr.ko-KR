# <a name="policy-error-when-sharing-a-calendar"></a>일정을 공유할 때의 정책 오류

일정을 공유 하려고 하는 경우 "정책에서이 수준에 하나 이상의 받는 사람에 게 사용 권한을 부여 하는 것을 허용 하지 않습니다." 라는 오류가 표시 되 면 [이 항목](https://support.microsoft.com/help/3187524/policy-does-not-allow-granting-permissions-at-this-level-to-one-or-mor)을 참조 하세요.
