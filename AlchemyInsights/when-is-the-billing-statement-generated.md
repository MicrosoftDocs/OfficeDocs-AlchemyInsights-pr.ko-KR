# <a name="when-is-the-billing-statement-generated"></a>When가 생성 된 대금 청구 문은 언제 입니까?

매달 Microsoft 365 관리 센터에서 새 청구 내역을 사용할 수 있음을 알리는 전자 메일을 받게 됩니다. 송장에 두 페이지가 포함 되어 있습니다. 1 페이지는 송장 요약으로, 송장, 주문, 기한, 지불 방법, 고객 지원에 문의 하는 방법에 대 한 일반 정보를 포함 합니다. 2 페이지는 각 구독의 청구 활동에 대 한 세부 정보를 포함 합니다.
  
대금 청구 개념에 대 한 자세한 내용은 [청구서 이해](https://support.office.com/article/0724b428-fb59-4962-8c37-6674166d7507)를 참조 하세요.
