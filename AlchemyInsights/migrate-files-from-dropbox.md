# <a name="migrate-files-from-dropbox"></a>Dropbox에서 파일 마이그레이션

Dropbox에서 SharePoint Online으로 마이그레이션하는 경우 **Mover** 를 사용해 보세요.

자세한 내용은 [Dropbox Connector 허용](https://docs.microsoft.com/sharepointmigration/mover-dropbox)을 참조하세요.
