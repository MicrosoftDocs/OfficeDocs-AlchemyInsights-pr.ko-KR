# <a name="setting-up-a-hybrid-environment"></a>하이브리드 환경 설정

협력 원활 하 게 관계 없이 자신의 데이터는 Office 365 또는 온-프레미스 서버에 저장 된 경우 사용자에 게, 하이브리드 환경 설정 해야할 수도 있습니다. 옵션에 대 한 개요를 가져오려면 [온-프레미스 환경와 Office 365의 통합](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx)을 체크아웃 합니다.
