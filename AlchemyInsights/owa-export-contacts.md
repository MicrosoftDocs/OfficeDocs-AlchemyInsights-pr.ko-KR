# <a name="export-contacts"></a>연락처 내보내기

1. 페이지 맨 아래에서 사용자 아이콘을 선택 하 여 사용자 페이지로 이동 합니다.
2. 도구 모음에서**연락처 내보내기** **관리** > 를 선택 합니다. 
3. 특정 폴더에서 모든 대화 상대 또는 연락처만 내보내려면 선택한 다음 **내보내기를**선택 합니다.
4. 페이지 맨 아래에서 **저장** 을 선택 하 여 기본 다운로드 폴더에 "contacts"를 저장 합니다. 또는 **저장** 옆에 있는 화살표를 선택 하 고 다른 **이름으로 저장**을 선택 합니다.

자세한 내용은 [Outlook에서 연락처 내보내기를](https://support.office.com/article/10f09abd-643c-4495-bb80-543714eca73f#ID0EAACAAA=Outlook_on_the_web)참조 하세요.
