outlook을 사용 하 여 영향을 받는 컴퓨터에서 [암호 진단을 요청](https://aka.ms/SaRA-OutlookPwdPrompt) 하는 것이 좋습니다. 이 [SaRA](https://diagnostics.office.com/#/) 진단은 자동 검사를 수행 하 고 검색 된 문제를 해결 하는 데 사용할 수 있는 가능한 솔루션을 반환 합니다.
