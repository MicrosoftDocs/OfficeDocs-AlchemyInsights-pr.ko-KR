# <a name="solutions-for-unsupported-operating-system"></a>"지원 하지 않는 운영 체제"에 대 한 솔루션



Office 365를 설치 하려고 하면 장치 [시스템 요구 사항](https://products.office.com/office-system-requirements)에 맞지 않는 경우에 **지원 되지않는 운영 체제** 오류 메시지가 나타날 합니다.
  
Office 365 필요합니다.
  
- PC: 10 Windows, Windows 8.1, Windows 8, Windows 7 서비스 팩 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 또는 Windows Server 2008 R2
    
    또는
    
- Mac OS X 10.10
    
모바일 장치 또는 프로그램 Chromebook에 Office를 설치 하는 동안이 메시지가 표시 됨을 [설치 하 고는 Chromebook에서 Microsoft Office를 실행 하는 방법을](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA)참조 합니다.
