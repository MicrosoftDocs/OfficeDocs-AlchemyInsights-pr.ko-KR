# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>메일 그룹을 Microsoft 365 그룹으로 변환

메일 그룹은 특정 기준을 충족 하는 경우 Microsoft 365 그룹으로 변환할 수 있습니다. 그룹을 변환 하거나 변환 문제를 해결 하려면 [Outlook에서 배포 목록을 Microsoft 365 그룹으로 업그레이드](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists)를 참조 하십시오.
