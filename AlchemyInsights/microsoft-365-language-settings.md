# <a name="microsoft-365-language-settings"></a>Microsoft 365 언어 설정

사용자 또는 전체 테 넌 트에 대 한 표시 언어 및 기타 설정을 변경 하려는 경우를 결정 합니다.

- 사용자의 경우 [모든 앱에 대해 비즈니스용 Microsoft 365에서 표시 언어 및 표준 시간대 변경을](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b)참조 하세요.
- 테 넌 트에 대 한 자세한 내용은  [Set the language and 국가별 설정 (Microsoft 365](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region))을 참조 하십시오.
