# <a name="microsoft-365-language-settings"></a>Microsoft 365 언어 설정

사용자 또는 전체 테넌트에 대해 표시 언어 및 기타 설정을 변경할지 여부를 확인합니다.

- 사용자의 경우 모든 앱에 대한 [비즈니스용 Microsoft 365에서 표시](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b)언어 및 표준 시간대 변경을 참조하세요.
- 테넌트에 대한 자세한 내용은  [Microsoft 365의](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region)언어 및 국가별 설정 설정을 참조하세요.
