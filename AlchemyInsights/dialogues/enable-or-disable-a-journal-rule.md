# <a name="enable-or-disable-a-journal-rule"></a>저널 규칙 사용 또는 사용 안 함

방법은 다음과 같습니다.

1. Exchange 관리 [센터에서](https://go.microsoft.com/fwlink/p/?linkid=2059104)규정 준수 관리 저널 **규칙으로**  >  **이동하십시오.**
2. 목록 보기의 규칙 이름 옆에 있는 **ON** 열에서 규칙을 사용하도록 설정하려면 확인란을 선택하거나 규칙을 사용하지 않도록 설정하려면 이 확인란을 선택 취소합니다.
