# <a name="use-your-browser-to-check-spelling"></a>브라우저를 사용 하 여 맞춤법을 검사 하려면

Outlook.com에서 맞춤법 검사 옵션이 없습니다. 맞춤법 검사를 실행 하려면 웹 브라우저를 사용 하 여 필요 합니다. 맞춤법 검사 하는 것은 Microsoft에 지, Internet Explorer 10 이상 버전, 및 현재 버전의 Firefox, Chrome 및 Safari에서 사용할 수 있습니다. 맞춤법을 검사 하는 방법에 대 한 자세한 내용은 웹 브라우저에 대 한 옵션을 확인 합니다.
