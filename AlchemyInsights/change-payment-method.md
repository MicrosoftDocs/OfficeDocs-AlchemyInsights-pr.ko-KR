# <a name="change-payment-method"></a>결제 방법 변경

**청구** \> **구독**으로 이동 하 여 구성 된 결제 방법을 확인할 수 있습니다. 확인할 구독을 선택합니다. **결제 방법**아래에 구독에 대해 구성 된 결제 방법이 표시 됩니다. 기존 결제 방법에 대 한 세부 정보를 업데이트 하려면 **결제 정보 업데이트**를 선택 합니다.
  
1. 관리 센터에서 **청구** \> [구독](https://go.microsoft.com/fwlink/p/?linkid=842054) 페이지로 이동 합니다.

2. 결제 방법을 변경 하려는 구독을 선택한 다음 **결제 방법 변경을**선택 합니다.

3. 화면에 나오는 단계에 따라 변경을 완료 합니다.

단계별 지침에 대해서는 다음을 참조 하십시오. [Office 365 비즈니스에 대 한 결제 방법 변경](https://docs.microsoft.com/en-us/office365/admin/subscriptions-and-billing/change-payment-method)
