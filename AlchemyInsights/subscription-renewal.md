# <a name="subscription-recurring-billing"></a>구독 되풀이 대금 청구

매월 또는 매년 구독 비용이 청구 되는 시기를 확인 하려는 경우 (되풀이 되는 **대금 청구** 가 설정 된 경우) 또는 만료 되는 경우 ( **되풀이** 비용이 해제 된 경우) 다음 단계를 수행 합니다.
  
1. 관리 센터에서 **결제**\> [내 상품](https://go.microsoft.com/fwlink/p/?linkid=842054)페이지로 이동하세요.
2. **제품** 탭에서 더 자세히 알아볼 구독을 찾습니다. **구독 상태** 열에는 갱신 또는 만료 날짜가 표시 됩니다.
3. 변경을 수행 하려면 **추가 작업** (점 3 개)을 선택한 다음 되풀이 되는 **청구 편집**을 선택 합니다.
4. 되풀이 되는 **청구 편집** 창에서 **설정**, **켜기, 한 번 갱신**또는 **해제**를 선택 합니다.
5. **저장**을 선택합니다.

자세한 내용은 [되풀이 청구 해제 또는 설정](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/renew-your-subscription)을 참조하세요.
