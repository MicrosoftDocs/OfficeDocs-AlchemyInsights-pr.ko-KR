# <a name="subscription-recurring-billing"></a>구독 되풀이 대금 청구

매월 또는 매년 구독 비용이 청구 되는 시기를 확인 하려면 (되풀이 되는 **청구** 를 설정 해야 하는 경우) 또는 만료 ( **되풀이** 비용이 해제 되 면):
  
1. 관리 센터에서 **Billing** \> **[제품](https://go.microsoft.com/fwlink/p/?linkid=842054)** 청구 페이지로 이동 합니다.

2. 보다 자세히 알고자 하는 구독의 경우 **대금 청구**에서 **갱신 설정** 또는 **만료** 예정 날짜를 참조 하세요.

4. 변경을 수행 하려면 **편집** 을 선택한 다음, 되풀이 되는 **청구** 설정 창에서 **해제** 를 선택 하 여 해제 합니다. 되풀이 청구가 꺼져 있는 경우, **월별**이나 **한 번**을 선택하여 켭니다.

자세한 내용은 [되풀이 청구 해제 또는 설정](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/renew-your-subscription)을 참조하세요.
