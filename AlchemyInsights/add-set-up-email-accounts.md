# <a name="problems-setting-up-an-email-account-in-outlook"></a>Outlook에서 전자 메일 계정을 설정 하는 문제

Outlook에서 Exchange 계정을 설정 하는 동안 문제가 발생 하는 경우 영향을 받는 컴퓨터에서 [outlook 계정 설정 도구](https://aka.ms/SaRA-OutlookSetupProfile) 를 실행할 수 있습니다. 다양 한 검사를 수행 하 고 계정 설정 문제에 대 한 가능한 해결 방법을 제공 합니다.
  
여전히 문제가 있는 경우 [Outlook에서 Exchange Online 사서함에 대해 Exchange 자동 검색을 사용 하 여 새 프로필을 설정할 수 없습니다](https://docs.microsoft.com/exchange/troubleshoot/outlook-profiles/cannot-set-up-profile-autodiscover).를 참조 하세요.
