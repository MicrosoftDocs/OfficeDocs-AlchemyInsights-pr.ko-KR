# <a name="trying-to-remove-your-domain"></a>도메인을 제거 하는 중 입니까?

[Office 365에서 도메인 제거](https://support.office.com/article/Remove-a-domain-from-Office-365-f09696b2-8c29-4588-a08b-b333da19810c.aspx)의 단계를 수행 합니다.
  
도메인을 제거 하기 전에 해당 도메인 이름에 할당 된 모든 사용자 또는 그룹을 업데이트 해야 합니다. 다른 일반적인 솔루션에 대 한 자세한 [도메인 FAQ](https://support.office.com/article/Domains-FAQ-1272bad0-4bd4-4796-8005-67d6fb3afc5a.aspx) 를 확인 하세요.
