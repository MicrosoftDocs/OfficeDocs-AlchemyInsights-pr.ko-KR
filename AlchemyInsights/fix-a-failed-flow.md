# <a name="fix-a-flow-that-failed"></a>실패 한 흐름 수정

1. [Microsoft Flow](https://flow.microsoft.com/)로 이동 하 여 오른쪽 위에 있는 알림 아이콘을 클릭 합니다. (또는 모바일 앱에서 활동 탭을 선택 합니다.) 세부 정보를 확인 하지 못한 흐름을 선택 합니다.
    
2. 흐름에 대 한 오류 메시지를 보려면 빨간색 느낌표 아이콘을 사용 하 여 해당 단계를 찾습니다.
    
3. 오류 메시지에 따라 흐름을 편집 하 고 수정할 수 있어야 합니다. 
    
[일반적인 흐름 오류](https://go.microsoft.com/fwlink/?linkid=872110)
