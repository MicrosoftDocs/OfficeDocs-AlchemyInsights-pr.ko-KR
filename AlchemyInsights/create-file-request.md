# <a name="how-to-create-a-file-request"></a>파일 요청을 만드는 방법

OneDrive의 파일 요청 기능을 사용 하면 보낸 링크를 사용 하 여 다른 사용자가 파일을 업로드할 수 있는 폴더를 선택할 수 있습니다. 파일을 요청 하는 사용자는 폴더에 있는 내용을 볼 수 없으며 파일을 업로드 하기만 하면 됩니다.
