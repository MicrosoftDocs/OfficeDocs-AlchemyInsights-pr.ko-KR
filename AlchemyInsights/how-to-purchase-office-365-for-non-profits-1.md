# <a name="microsoft-365-for-nonprofits---approved"></a>비 이익 승인 용 Microsoft 365

Microsoft 자격이 있는 비영리 단체의 상태가 승인 되었습니다. Microsoft 365 관리 센터에서 구매를 완료할 수 있습니다.

1. 관리 센터에서 **결제** \> [구매 서비스](https://go.microsoft.com/fwlink/p/?linkid=868433) 페이지로 이동 합니다.

2. 비영리 **계획**아래 목록에서 선택 합니다.

3. 구매할 구독을 가리키고 **지금 구입**을 선택 합니다.

자세한 내용은 비영리 또는 [연락처 판매](https://www.microsoft.com/nonprofits/contact-us) [에 대 한 Microsoft 365 resources](https://www.microsoft.com/nonprofits/microsoft-365) 를 참조 하세요.
