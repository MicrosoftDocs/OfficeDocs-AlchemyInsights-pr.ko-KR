# <a name="add-a-password-to-a-word-document"></a>Word 문서에 암호 추가

1. **파일**정보로 이동 하  >  **Info**  >  **Protect Document**  >  **여 암호를 사용 하 여 문서 암호화**를 보호 합니다.
2. 사용할 암호를 입력 한 다음 **확인**을 선택 합니다.
3. 암호를 다시 입력 한 다음 **확인**을 선택 합니다.
