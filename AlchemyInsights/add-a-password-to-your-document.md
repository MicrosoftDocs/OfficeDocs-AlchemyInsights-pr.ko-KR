# <a name="add-a-password-to-a-word-document"></a>Word 문서에 암호 추가

1. 파일 **정보**  >    >  **암호로 문서 암호화**  >  **보호로 이동하세요.**
2. 사용할 암호를 입력하고 확인 을 **선택 합니다.**
3. 암호를 다시 입력한 다음 확인 을 **선택 합니다.**
