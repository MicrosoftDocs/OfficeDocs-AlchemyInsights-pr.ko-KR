# <a name="close-your-outlookcom-account"></a>Outlook.com 계정 닫기

1. [계정 닫기](https://go.microsoft.com/fwlink/p/?linkid=845493) 페이지로 이동 합니다.
2. 페이지의 지시에 따라 계정을 닫을 준비가 되었는지 확인 하 고 **다음**을 선택 합니다.
3. 목록을 읽고 각 항목을 읽은 후 승인 하는 확인란을 선택 합니다.
4. **이유 선택** 드롭다운 목록에서 계정을 닫는 이유를 선택 합니다.
5. **폐쇄 계정 표시를**선택 합니다.

자세한 내용은 [Outlook.com 계정을 닫으세요](https://support.office.com/article/564b801e-2a47-4cb2-afa8-12ead3185038?wt.mc_id=Office_Outlook_com_Alchemy).를 참조 하십시오.
