# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Microsoft Edge 레거시 정책과 Microsoft Edge 정책 간의 매핑에 대한 자세한 정보

Microsoft Edge 레거시(버전 45 이전)의 그룹 정책과 MDM(모바일 장치 관리) 설정은 Microsoft Edge 버전 80의 관련 정책에 매핑됩니다. 자세한 내용은 [Microsoft Edge 레거시와 Microsoft Edge 정책 매핑](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge)을 참조하세요.
