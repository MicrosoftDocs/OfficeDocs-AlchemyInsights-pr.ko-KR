# <a name="yammer---possible-service-issue"></a>Yammer-가능한 서비스 문제

Yammer에 액세스할 때 사용자에 게 오류가 발생할 수 있는 일시적인 문제가 있을 수 있습니다. [서비스 상태 대시보드가](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) 네트워크에 영향을 주는지 확인 합니다.
