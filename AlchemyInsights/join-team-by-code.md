# <a name="join-a-team-by-code"></a>코드로 팀 참가

Microsoft는 사용자가 **팀 가입** 코드를 만들거나 **코드별 가입** 기능을 사용하여 팀에 가입할 수 없는 문제를 알고 있습니다. 해결하려면 팀 소유자가 필요한 경우 수동으로 사용자를 추가할 수 있습니다.
