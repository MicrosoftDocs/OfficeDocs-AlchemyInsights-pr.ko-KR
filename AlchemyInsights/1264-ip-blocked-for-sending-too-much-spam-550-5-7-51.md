# <a name="access-denied-banned-sender"></a>보낸 사람이 금지 액세스 거부

 **오류**: 명령, 호스트를 RCPT에 대 한 원격 서버에서 SMTP 오류: *이며* -com.mail.protection.outlook.com ( *xxx.xxx.xxx.xxx* ) 이유: 550 5.7.511 거부 된, 금지 된 보낸 사람이 [ *xx.xxx.x.xxx* ]에 액세스 합니다. 
  
Office 365 받는 사람에 게 전자 메일을 보낼 때이 오류가 발생 하면 영향을 받는 계정에서 수신된 거부 목록에서 제거 [https://sender.office.com](https://sender.office.com)합니다.
