# <a name="add-a-sharepoint-calendar"></a>SharePoint 달력 추가

SharePoint 일정은 때때로 Microsoft 365 그룹 달력과 혼동 되는 클래식 목록 유형입니다.
 
SharePoint 일정을 만들려면 다음 단계를 수행 합니다.
 
1.  **설정** 기어 아이콘을 선택 하 고 **앱 추가**를 선택 합니다.
2.  앱 검색 **찾기** 상자에 **일정** 을 입력 하 고 enter 키를 누릅니다.
3.  **달력**이라는 앱을 선택한 다음 일정의 이름을 입력 합니다.

Microsoft 365 그룹에 대 한 자세한 내용은 [microsoft 365 그룹에 대 한](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)자세한 정보를 참조 하십시오.
