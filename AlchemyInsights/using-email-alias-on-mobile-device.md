# <a name="email-aliases-on-mobile-devices"></a>모바일 장치의 전자 메일 별칭

전자 메일 별칭은 사서함에 구성 된 또 다른 전자 메일 주소 이므로 사용자의 별칭으로 전송 되는 메시지는 해당 사서함으로 배달 되 고 모바일 장치에서 읽을 수 있게 됩니다. 추가 구성은 필요 하지 않습니다.

그러나 사용자가 메시지에 회신 하거나 별칭을 보낸 사람 주소로 사용 하 여 새 메시지를 보낼 수는 없습니다. 보내는 모든 메시지는 기본 전자 메일 주소에서 전송 됩니다.
