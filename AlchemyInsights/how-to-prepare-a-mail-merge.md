# <a name="how-to-prepare-a-mail-merge"></a>편지 병합을 준비 하는 방법

다른 플랫폼의 편지 병합에 대 한 자세한 내용은 [대량 전자 메일, 편지, 레이블 및 봉투에 대해 편지 병합 사용](https://support.office.com/article/use-mail-merge-for-bulk-email-letters-labels-and-envelopes-f488ed5b-b849-4c11-9cff-932c49474705)의 지침을 따르세요.
 
Excel을 사용 하 여 편지 병합을 수행 하는 방법에 대 한 자세한 내용은 excel [스프레드시트를 사용 하 여](https://support.office.com/article/Mail-merge-using-an-Excel-spreadsheet-858c7d7f-5cc0-4ba1-9a7b-0a948fa3d7d3)편지 병합 및 편지 병합을 [위한 Excel 데이터 원본 준비](https://support.office.com/article/prepare-your-excel-data-source-for-a-word-mail-merge-2d802b6b-a3a3-43e5-bb76-2cac7c68673e) 를 참조 하세요.
