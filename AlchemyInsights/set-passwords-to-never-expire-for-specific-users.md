# <a name="set-passwords-to-never-expire"></a>만료 되지 않도록 설정 암호

모든 암호 만료 되지 않도록 설정 하려면로 이동 [보안 &amp; 개인정보](https://portal.office.com/adminportal/home#/settings/security)합니다. **암호 정책**, 옆에 있는 **편집** 을 클릭 하 고 **전환**슬라이더를 설정 합니다.
  
자세한 내용은 참조: [조직에 대 한 암호 만료 정책을 설정 합니다.](https://support.office.com/article/0f54736f-eb22-414c-8273-498a0918678f)
