# <a name="set-passwords-to-never-expire"></a>암호가 만료 되지 않도록 설정

모든 암호가 만료 되지 않도록 설정 하려면 [ &amp; 보안 개인 정보](https://portal.office.com/adminportal/home#/settings/security)로 이동 합니다. **암호 정책**옆에 있는 **편집** 을 클릭 하 고 슬라이더를 **켜**세요.
  
자세한 내용은 [조직에 대 한 암호 만료 정책 설정을](https://support.office.com/article/0f54736f-eb22-414c-8273-498a0918678f) 참조 하십시오.
  
[특정 사용자에 대해 암호가 만료 되지 않도록 설정](https://support.office.com/article/f493e3af-e1d8-4668-9211-230c245a0466)
