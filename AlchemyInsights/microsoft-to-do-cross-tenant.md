# <a name="microsoft-to-do-cross-tenant-sharing"></a>Microsoft를 통해 교차 테 넌 트 공유

Microsoft는 개인 계정과 회사 계정 간의 목록 공유를 지원 합니다. 이렇게 하면 조직의 사용자가

개인 Microsoft 계정이 소유한 목록에 연결 하 고, 보고, 수정 하 고, 데이터를 추가 합니다.

관리자는이 옵션을 선택 취소 하 여이 기능을 사용 하지 않도록 설정할 수 있습니다.

관리자는이 설정을 **Settings**  >  **조직 설정**  >  **서비스**  >  **Microsoft** 의 설정 아래에 있는 관리 포털에서 찾을 수 있습니다.  

[공유 목록에 참가할 수 없는 이유는 무엇 인가요?](https://support.microsoft.com/office/why-can-t-i-join-a-shared-list-3a6195de-e3a8-437a-b562-7c8c011dc574?ui=en-us&rs=en-us&ad=us)
