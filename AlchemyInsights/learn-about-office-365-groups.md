# <a name="learn-about-office-365-groups"></a>Office 365 그룹에 대 한 자세한 정보

Office 365 그룹은 이미 사용 중인 Office 365 도구에서 작동 하는 서비스 이므로 문서를 작성 하거나, 스프레드시트를 작성 하 고, 프로젝트 계획을 작업 하거나, 모임을 예약 하거나, 전자 메일을 보낼 때 동료와 공동으로 작업할 수 있습니다. 자세한 내용은 [Office 365 그룹에 대 한](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2)정보를 참조 하세요.
