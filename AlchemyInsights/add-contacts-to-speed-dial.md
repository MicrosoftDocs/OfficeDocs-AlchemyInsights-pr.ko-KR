# <a name="add-contacts-to-speed-dial"></a>빠른 전화 걸기에 연락처 추가

1. **통화**중 **연락처로**이동 합니다.
2. 빠른 전화 걸기에 추가할 사용자의 카드가 표시 되 면 카드에서  **기타 옵션**  (세 점)을 선택 하 고  **빠른 전화 걸기에 추가를**선택 합니다.

사용자가 표시 되지 않으면 화면 오른쪽 위에 있는  **빠른 전화 걸기 추가**  를 선택 하 고 이름을 입력 한 다음 사용자를 선택 하 고  **추가**를 선택 합니다.
