# <a name="working-with-sharepoint-permission-levels"></a>SharePoint 사용 권한 수준 사용

SharePoint에는 **참가** 및 **보기 전용**과 같이 사용할 수 있는 기본 사용 권한 수준이 제공 됩니다. 그러나 이러한 방법 중 하나가 요구 사항을 충족 하지 않으면 새 사용 권한 수준을 만들 수 있습니다. 기본값을 변경 하지 않으면 사용 권한 수준을 쉽게 추적할 수 있습니다.

[사용 권한 수준을 만들고 편집하는 방법](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
