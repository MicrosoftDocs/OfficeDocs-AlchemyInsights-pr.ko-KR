# <a name="add-contacts-to-an-office-365-group"></a>Office 365 그룹에 대화 상대를 추가 합니다.

그룹 소유자는 외부 사용자를 추가 하 고 다음이 지침을 수행 하 여 Outlook의 연락처를 메일 수: [Outlook에서 그룹 구성원 추가 및 제거](https://support.office.com/article/3b650f4a-5c9b-4f94-a1bb-0cca4b1091de?wt.mc_id=add_contacts_group.aspx)
  
또한 추가 하거나 관리자 포털에서 그룹 구성원을 제거할 수: [Office 365 관리 센터를 사용 하 여 Office 365 그룹에서 구성원을 추가 또는 제거](https://support.office.com/article/e186d224-a324-4afa-8300-0e4fc0c3000a?wt.mc_id=add_guest_portal.aspx)
