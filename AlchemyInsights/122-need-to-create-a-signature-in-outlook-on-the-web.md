# <a name="need-to-create-an-email-signature"></a>전자 메일 서명을 만들 필요가 있습니까?

Outlook 계정 또는 웹 (Office 365 작업 또는 학교) 계정에는 Outlook을 설치한 경우 서명을 만들 수 및 모든 보내는 메시지에 자동으로 추가 또는 선택적으로 특정 보내는 메시지에 추가 합니다. 전자 메일 서명을 Outlook 및 웹에 있는 Outlook에서 서로 다른 인지 참고 합니다. 전자 메일 서명을 만들려면 참조 [만들기 Outlook에서 메시지에 서명을 추가 하 고](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) 또는 [만들기 웹에 있는 Outlook에서 전자 메일 서명을 추가 하 고](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)합니다.
