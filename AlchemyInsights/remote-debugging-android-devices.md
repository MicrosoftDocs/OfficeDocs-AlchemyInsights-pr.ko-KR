# <a name="use-microsoft-edge-to-remotely-debug-an-android-device-from-a-development-computer"></a>Microsoft Edge를 사용하여 개발 컴퓨터에서 원격으로 Android 장치 디버그

컴퓨터에서 Android 장치에서 라이브 콘텐츠를 원격으로 디버그하려면

1. 원격 디버깅을 위해 Android 장치를 설정하고 개발 컴퓨터에서 검색합니다.
2. 개발 컴퓨터에서 Android 장치에서 라이브 콘텐츠를 검사하고 디버그합니다.
3. Android 장치에서 개발 컴퓨터의 DevTools 인스턴스로의 스크린캐스트 라이브 콘텐츠
