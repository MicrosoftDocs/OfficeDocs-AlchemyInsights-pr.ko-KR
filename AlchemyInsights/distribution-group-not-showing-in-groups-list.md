# <a name="distribution-group-not-showing-in-groups-list"></a>그룹으로 표시 되지 않는 메일 그룹 목록

새 메일 그룹이 관리 센터에 표시 되려면 최대 60 분이 소요 될 수 있습니다.
  
지금 구성원을 추가 하거나 다른 편집 작업을 수행 하려면 [Exchange 관리 센터](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1)에서 그룹 기능을 사용 하세요.
