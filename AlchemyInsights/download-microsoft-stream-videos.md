# <a name="download-microsoft-stream-videos"></a>Microsoft Stream 비디오 다운로드하기

**비디오 파일을 업로드**했거나 **비디오의 소유자**인 경우에는 원래 업로드한 비디오를 **다운로드**할 수 있습니다.

1. Stream에서 다운로드하려는 비디오를 찾습니다.

2. 기타 메뉴 아이콘 *...* 에서 기타 메뉴 아이콘을 클릭합니다.

3. **원본 비디오 다운로드**를 선택합니다.
