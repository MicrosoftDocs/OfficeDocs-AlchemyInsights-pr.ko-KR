# <a name="web-only-limited-access"></a>Web-Only 제한 된 액세스

Web-Only 제한 된 액세스는 웹 개체에 대 한 사용자 액세스를 허용 하는 ' 제한적인 액세스 ' 사용 권한 수준의 변형입니다.

자세한 내용은 [기본 사용 권한 수준을](https://docs.microsoft.com/sharepoint/understanding-permission-levels#default-permission-levels)참조 하십시오.
