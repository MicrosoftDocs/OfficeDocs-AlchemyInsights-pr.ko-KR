# <a name="limit-sharing-in-microsoft-365"></a>Microsoft 365에서의 공유 제한

내부 공유를 완전히 사용하지 않도록 설정하거나 사이트에서 공유 단추를 제거할 수 있는 것은 아니며, 조직의 요구를 충족하기 위해 Microsoft 365에서 공유를 제한할 수 있는 다양한 방법이 있습니다. 자세한 내용은 [Microsoft 365에서 공유 제한을 참조하세요.](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)
