# <a name="how-to-add-or-reduce-licenses"></a>추가 하거나 라이선스를 축소 하는 방법

제거 또는 추가 라이센스를 구입 하는 방법을 다음과 같습니다.
  
1. 관리 센터에서 [구독](https://go.microsoft.com/fwlink/p/?linkid=842054) 페이지로 이동하거나 **청구** \> **구독**을 선택합니다.
    
2. 왼쪽에서 라이선스 추가/제거를 원하는 구독을 선택 합니다.
    
3. **라이선스 추가/제거**를 선택합니다.
    
4. 필요한 사용자의 총 수를 입력 합니다.
    
    예, 현재 라이선스 25 개를 포함 하는 경우 5를 제거 하려면이 필드에는 20을 입력 합니다.
    
5. **Submit(전송)** 을 선택합니다.
    
> [!NOTE]
> 모든 새 무료 (라이선스 증가) 또는 신용 (라이선스 감소) 다음 청구서에 표시 됩니다.
