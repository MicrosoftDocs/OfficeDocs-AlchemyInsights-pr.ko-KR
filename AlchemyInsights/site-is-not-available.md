# <a name="site-is-not-available"></a>사이트를 사용할 수 없는 경우

OneDrive 또는 SharePoint 사이트에 여러 사용자에 게 사용할 수 없는 경우 임시 서비스 문제가 있을 수 있습니다. 테 넌 트의 영향을 확인 하려면[서비스 상태 대시보드](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) 를 확인 합니다.
