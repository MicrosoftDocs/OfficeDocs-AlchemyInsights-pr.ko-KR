# <a name="site-is-not-available"></a>사이트를 사용할 수 없음

SharePoint 또는 OneDrive 사이트를 여러 사용자가 사용할 수 없는 경우 일시적인 서비스 문제가 있을 수 있습니다. [서비스 상태 대시보드](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) 를 검사 하 여 테 넌 트가 영향을 받는 것인지 확인 합니다.
