# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Mac에서 OneDrive 또는 SharePoint 파일 설치, 설정 및 동기화 

Mac [OS X에서 onedrive 동기화 클라이언트를 사용 하 여 파일을 동기화](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f)하는 지침을 사용 하 여 Mac에 Onedrive 또는 SharePoint 파일을 설치, 설정 및 동기화 합니다.

OneDrive 앱이 설치 되어 있고 Mac에 설치 되어 있지만 사용자에 게 동기화 문제가 발생 하는 경우 [동기화 문제 해결 OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&amp;rs=en-US&amp;ad=US)를 참조 하세요.

> [!NOTE]
> Windows 용 OneDrive 동기화에서 알려진 폴더 이동 기능을 Mac에서는 사용할 수 없습니다.
