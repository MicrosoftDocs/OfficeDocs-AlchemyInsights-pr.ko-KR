# <a name="public-folder-access-from-mobile-devices"></a>모바일 장치에서 든 지 공용 폴더 액세스

공용 폴더 모바일 장치에서 액세스할 수 없습니다.
  
공용 폴더와 유사한 기능을 포함 하 고 모바일 장치에서 액세스 하려는 경우 [Office 365 그룹에 대 한 설명](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) 더 나은 대체에 대 한 참조입니다.
