# <a name="public-folder-access-from-mobile-devices"></a>모바일 장치에서의 공용 폴더 액세스

모바일 장치에서는 공용 폴더에 액세스할 수 없습니다.
  
공용 폴더와 유사한 기능을 사용 하 고 모바일 장치에서 액세스 하려면 [Office 365 그룹에 대 한](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) 자세한 내용은 더 나은 대안을 참조 하십시오.
