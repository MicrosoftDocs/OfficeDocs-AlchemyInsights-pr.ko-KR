# <a name="issues-while-migrating-data-to-sharepoint-online"></a>데이터를 SharePoint Online으로 마이그레이션하는 동안 문제가 발생 합니다.

<p>100TB의 데이터를 Sharepoint Online으로 마이그레이션하는 것으로 보입니다.</p> <ol> <li>아래의 전화 &ldquo;/전자 메일로&rdquo; 새 서비스 요청을 클릭 하 고 쿼리를 텍스트 &ldquo;상자에 100tb&rdquo; 를 초과 하는 Sharepoint로 마이그레이션합니다.</li> <li>티켓이 전송 된 후에는 다음 정보를 사용 하 여 업데이트 하세요. <ul> <li>회사 이름 및 테넌트 URL</li> <li>예상 마이그레이션 크기</li> <li>예상 마이그레이션 시작 및 완료 시기</li> <li>마이그레이션하는 콘텐츠의 위치(예: SharePoint Server, Box, GDrive, 파일 공유 등)</li> </ul> </li> </ol>
