# <a name="workflows-in-sharepoint"></a>SharePoint의 워크플로

<p>SharePoint 워크플로에서 전자 메일을 보내지 않는 경우 조직에서 Exchange Online 보낸 사람 제한이 발생 했을 수 있습니다.&nbsp;</p> <p>' 워크플로 일시 중단 ' 오류 메시지가 다음 항목 중 하나에 해당 하는 경우 발생할 수 있습니다.</p> <ul> <li>Sharepoint Online의 워크플로를 사용 하 2013 2010 고 있으며,이를 수행 하는 사용자</li> <li>워크플로는 한 번에 200 명 이상의 사용자에 게 사용자 지정 전자 메일 메시지를 전송 하거나, 하루에 1만 명의 받는 사람을 초과 하거나, 분당 30 개 이상의 메시지를 보내도록 구성 됩니다.</li> <li>워크플로를 실행 하는 경우 전자 메일 메시지가 전송 되지 않으며, <strong>내부 상태가 일시 중단</strong> 됨 또는 <strong>받는 사람에 게 보낼 수 없음</strong> 으로 설정 됩니다.</li> </ul> <p>자세한 내용은 다음 <a href="https://support.office.com/en-us/article/-daily-email-limit-has-exceeded-and-your-workflow-has-been-suspended-or-unable-to-send-to-a-recipient-error-in-a-sharepoint-online-workflow-89d02169-5fa6-4259-affc-73edb6ca9fb6?ui=en-US&amp;rs=en-US&amp;ad=US">문서</a>를 참조 하세요.</p>
